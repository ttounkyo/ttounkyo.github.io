Selectores en CSS

Los selectores en CSS indican a qué elementos HTML se aplicarán las reglas de estilo. A continuación se muestran los selectores más comunes con ejemplos.

1. Selector de tipo o etiqueta

Selecciona todos los elementos de un tipo determinado.

p {
    color: red;
}

Ejemplo: Aplica color rojo a todos los párrafos <p>.

2. Selector de clase

Selecciona elementos que tienen un atributo class específico. Se indica con un punto (.).

.destacado {
    background-color: yellow;
}
<p class="destacado">Texto importante</p>

3. Selector de ID

Selecciona un único elemento con un id específico. Se indica con una almohadilla (#).

#titulo {
    font-size: 24px;
}
<h1 id="titulo">Bienvenidos</h1>

4. Selector universal

Selecciona todos los elementos del documento.

* {
    margin: 0;
    padding: 0;
}

Ejemplo: Se usa para eliminar márgenes y rellenos por defecto.

5. Selector de descendiente

Selecciona elementos que estén dentro de otro elemento específico.

article p {
    color: green;
}

Ejemplo: Aplica color verde a los <p> que estén dentro de un <article>.

6. Selectores combinados y avanzados

  • h1, h2: selecciona varios elementos a la vez.
  • a:hover: selecciona cuando el usuario pasa el ratón por encima.
  • ul > li: selecciona solo los elementos hijos directos.
a:hover {
    text-decoration: underline;
}

Ejemplo: Subraya los enlaces al pasar el ratón.
