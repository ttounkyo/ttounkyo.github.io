Elementos más importantes de CSS

CSS se compone de reglas que se aplican a elementos HTML para definir su estilo visual. Cada regla tiene uno o más selectores y un conjunto de declaraciones formadas por propiedades y valores.

Estructura de una regla CSS

selector {
    propiedad: valor; /* Declaración */
    propiedad: valor; /* Declaración */
}

Ejemplo:

p {
    color: red;
} /* → Aplica el color rojo a todos los párrafos. */

Propiedades fundamentales de CSS

Propiedad Descripción Ejemplo
color Define el color del texto. color: blue;
background-color Color de fondo del elemento. background-color: yellow;
font-size Tamaño del texto. font-size: 16px;
font-family Define la fuente del texto. font-family: Arial, sans-serif;
font-weight Grosor de la fuente. font-weight: bold;
font-style Estilo de la fuente. font-style: italic;
text-decoration Decoración del texto (subrayado, tachado, etc.). text-decoration: underline;
margin Espacio exterior del elemento. margin: 10px;
padding Espacio interior (entre el contenido y el borde). padding: 15px;
border Define el borde del elemento. border: 1px solid black;
width Ancho del elemento. width: 200px;
height Alto del elemento. height: 100px;
display Tipo de visualización del elemento (block, inline, inline-block, flex, grid, etc.). display: block;
float Posiciona un elemento a la izquierda o derecha de su contenedor, permitiendo que otros elementos floten a su alrededor. float: left;
position Método de posicionamiento de un elemento (static, relative, absolute, fixed, sticky). position: relative;
top, right, bottom, left Se utilizan con position para especificar el desplazamiento del elemento. top: 20px;

Unidades de Medida en CSS

En CSS, las unidades de medida se utilizan para especificar la longitud, el tamaño y otras dimensiones de los elementos. Es importante comprender las diferentes unidades disponibles para lograr el diseño deseado y adaptabilidad.

Unidades Absolutas

Las unidades absolutas representan un valor fijo y no varían según el entorno. Sin embargo, en el contexto web, la más común es el píxel (px), aunque las otras son estándares físicos.

  • px (píxel): Es la unidad más común en la web. Representa un punto en la pantalla del dispositivo. 
    font-size: 16px;
  • pt (punto): Tradicionalmente usado en imprenta (1 punto = 1/72 de pulgada). No es tan común en la web para la visualización en pantalla. 
    font-size: 12pt;
  • cm (centímetro), mm (milímetro), in (pulgada): Unidades físicas. Raramente se usan para diseño web, ya que el tamaño de la pantalla varía. 
    width: 10cm;

Unidades Relativas

Las unidades relativas basan su valor en otro valor, lo que las hace muy útiles para crear diseños flexibles y adaptables.

  • em: Relativo al tamaño de la fuente del elemento padre. Si el padre tiene un font-size de 16px, entonces 1em será 16px en el hijo (a menos que el hijo también defina su propio font-size). 
    .padre { font-size: 16px; }
.hijo { font-size: 1.5em; /* → 24px (1.5 * 16) */ }
  • rem (root em): Relativo al tamaño de la fuente del elemento raíz (<html>). Esto ayuda a mantener una escala de fuente consistente en toda la página. 
    html { font-size: 10px; }
body { font-size: 1.6rem; /* → 16px (1.6 * 10) */ }
  • %: Porcentaje. A menudo relativo al tamaño del elemento padre. Por ejemplo, el width: 50%; hará que un elemento ocupe la mitad del ancho de su contenedor padre. 
    .contenedor { width: 400px; }
.elemento { width: 50%; /* → 200px */ }
  • vw (viewport width): Relativo al 1% del ancho de la ventana del navegador. 100vw es el ancho completo de la ventana. 
    .ancho-completo { width: 100vw; }
  • vh (viewport height): Relativo al 1% de la altura de la ventana del navegador. 100vh es la altura completa de la ventana. 
    .alto-completo { height: 100vh; }
  • vmin (viewport minimum): Relativo al 1% de la dimensión más pequeña de la ventana (ancho o alto). 
    .cuadrado-minimo { width: 50vmin; height: 50vmin; /* Siempre será un cuadrado basado en la dimensión más pequeña */ }
  • vmax (viewport maximum): Relativo al 1% de la dimensión más grande de la ventana (ancho o alto). 
    .elemento-maximo { font-size: 10vmax; /* La fuente crecerá según la dimensión más grande */ }

Consideraciones

La elección de la unidad de medida depende del contexto y del efecto que se quiera lograr. Las unidades relativas son generalmente preferibles para crear diseños adaptables que funcionen bien en diferentes dispositivos y tamaños de pantalla. Los rem son especialmente útiles para mantener una escala de texto consistente, mientras que las unidades de viewport (vw, vh, vmin, vmax) son ideales para diseños que ocupan toda la ventana del navegador.

Ejemplo práctico con varias propiedades

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Estilos CSS básicos</title>
    <style>
        .caja {
            background-color: #e0f7fa;
            color: #006064;
            font-size: 18px;
            padding: 20px;
            margin: 15px 0;
            border: 2px dashed #0097a7;
            text-align: center;
        }
    </style>
</head>
<body>
    <div class="caja">
        ¡Hola! Soy una caja estilizada con CSS 😊
    </div>
</body>
</html>

Este ejemplo utiliza varias propiedades básicas de CSS para dar estilo a un <div>, incluyendo color, fondo, bordes y espaciado.

La Jerarquía (Especificidad) en CSS

Cuando varios estilos intentan aplicarse al mismo elemento, el navegador sigue una jerarquía para determinar cuál prevalece. Esta jerarquía se basa en la especificidad de los selectores.

De manera general, los estilos se aplican en el siguiente orden de menor a mayor prioridad:

  1. Estilos por defecto del navegador: Son los estilos básicos que el navegador aplica a los elementos si no se especifica ningún otro.
  2. Estilos en archivos CSS externos (enlazados): Los estilos definidos en archivos .css enlazados en la sección <head> del HTML.
  3. Estilos internos (en la etiqueta <style>): Los estilos definidos directamente dentro de la etiqueta <style> en el documento HTML.
  4. Estilos en línea: Los estilos aplicados directamente a un elemento HTML usando el atributo style. Estos tienen la mayor prioridad.

Selectores y Especificidad:

Dentro de los estilos en archivos CSS o internos, la especificidad de los selectores también juega un papel importante. Aquí tienes una idea general de la prioridad:

  1. Selectores de tipo (elementos): Como p, h1, div. Son los menos específicos.
  2. Selectores de clase: Como .container, .text-primary. Son más específicos que los selectores de tipo.
  3. Selectores de ID: Como #mi-elemento. Son aún más específicos que los selectores de clase.
  4. Estilos en línea: (Mencionados anteriormente) Tienen la mayor especificidad.
Ejemplo para ilustrar la especificidad:
<div id="miDiv" class="contenedor texto-rojo" style="color: green;">Este texto tiene varios estilos.</div>
/* Estilo para todos los div */
div {
    color: blue;
}

/* Estilo para la clase texto-rojo */
.texto-rojo {
    color: red;
}

/* Estilo para el ID miDiv */
#miDiv {
    color: orange;
}

En este ejemplo, el texto dentro del <div> aparecerá en verde. Esto se debe a que el estilo en línea (style="color: green;") tiene la mayor prioridad. Si elimináramos el estilo en línea, el texto sería naranja, ya que el selector de ID (#miDiv) es más específico que el selector de clase (.texto-rojo), que a su vez es más específico que el selector de tipo (div).
