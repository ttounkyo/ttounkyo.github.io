Selectores descendientes y combinadores en CSS

Los combinadores en CSS permiten definir relaciones entre elementos HTML. Son esenciales para aplicar estilos específicos según la posición de los elementos en el DOM.

Selector Nombre Relación Ejemplo
elemento1 elemento2 Descendiente Selecciona todos los elemento2 dentro de elemento1 (a cualquier nivel) div p → Todos los <p> dentro de <div>
elemento1 > elemento2 Hijo directo Solo los hijos directos de elemento1 ul > li → Solo <li> hijos directos de <ul>
elemento1 + elemento2 Hermano adyacente Selecciona el elemento2 que esté justo después de elemento1 h1 + p → El primer <p> tras un <h1>
elemento1 ~ elemento2 Hermanos generales Selecciona todos los elemento2 que estén después de elemento1, como hermanos h2 ~ p → Todos los <p> tras un <h2>
* Universal Selecciona todos los elementos * { margin: 0; } → Reinicia márgenes de todos los elementos

Ejemplo práctico combinado

<style>
    div p {
        color: blue;
    }

    div > p {
        font-weight: bold;
    }

    h2 + p {
        background-color: yellow;
    }

    h2 ~ p {
        font-style: italic;
    }

    * {
        box-sizing: border-box;
    }
</style>

<div>
    <p>Soy hijo directo y descendiente de un div.</p>
    <section>
        <p>Soy un descendiente, pero no hijo directo.</p>
    </section>
</div>

<h2>Título</h2>
<p>Soy hermano adyacente del h2.</p>
<p>Soy hermano general del h2.</p>

Este ejemplo muestra cómo aplicar distintos estilos según la relación estructural entre elementos. Puedes visualizar cómo se comporta cada selector en un navegador para entenderlo mejor.
