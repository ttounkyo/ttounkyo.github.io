Propiedades Fundamentales de CSS

En CSS, podemos definir diversas propiedades para los elementos HTML, y a cada propiedad asignarle un valor específico. Por ejemplo, la propiedad color (color del texto) puede tener valores como red, blue, #FF0000, etc.

A continuación, exploraremos algunas de las propiedades más importantes de CSS, cómo se utilizan y los valores que pueden tomar. El objetivo es demostrar cómo puedes controlar el aspecto visual de tus elementos web principalmente a través de estilos CSS, manteniendo tu HTML más enfocado en la estructura.

Cubriremos las siguientes propiedades:

  • Fuentes y Texto
  • Color
  • Fondo (Background)
  • Espaciado (Margin y Padding)
  • Dimensiones (Ancho)
  • Borde (Border)
  • Posicionamiento y Visualización de Elementos

Fuentes y Texto

Estas propiedades permiten controlar la apariencia visual del texto dentro de los elementos HTML, incluyendo párrafos, encabezados y enlaces. Las propiedades más comunes son:

font-family

Define la fuente tipográfica para el texto. Se recomienda especificar una lista de fuentes, separadas por comas, para que el navegador pueda usar una fuente alternativa si la primera no está disponible en el sistema del usuario. La lista debe terminar con una familia genérica (serif, sans-serif, monospace, cursive, fantasy). 

.ejemplo-fuente {
    font-family: "Arial", "Verdana", sans-serif;
}

Si el usuario no tiene Arial o Verdana, se usará cualquier fuente sans-serif disponible en su sistema.

font-style

Especifica si el texto debe mostrarse en cursiva. Los valores comunes son normal y italic. 

.ejemplo-cursiva {
    font-style: italic;
}

font-weight

Define el grosor o "negrita" de la fuente. Los valores comunes incluyen normal, bold, y valores numéricos de 100 a 900 (en múltiplos de 100). bold es equivalente a 700. 

.ejemplo-negrita {
    font-weight: bold;
}

.ejemplo-fino {
    font-weight: 300; /* Requiere que la fuente soporte este grosor */
}

font-size

Establece el tamaño de la fuente. Es crucial elegir unidades adecuadas para la web, como px, em, rem, o %, que permitan adaptabilidad. Las unidades absolutas como pt, cm, etc., no son ideales para la visualización en pantalla ya que no se escalan con la configuración del usuario o el zoom del navegador de forma predecible (Ver sección "Unidades de Medida" del ejemplo anterior para más detalle). 

.ejemplo-tamano {
    font-size: 16px; /* Tamaño fijo en píxeles */
}

.ejemplo-relativo {
    font-size: 1.2rem; /* 1.2 veces el tamaño de fuente del elemento raíz */
}

Propiedad abreviada font

Esta es una propiedad abreviada que permite establecer varias propiedades de fuente en una sola declaración. El orden es importante y generalmente sigue este patrón (los valores entre corchetes son opcionales): [font-style] [font-weight] font-size font-family. 

.ejemplo-font-shorthand {
    font: italic bold 16px "Arial", sans-serif;
}

.ejemplo-font-shorthand2 {
    font: 1em sans-serif; /* Solo tamaño y familia, el resto por defecto */
}

text-decoration

Se usa para añadir decoraciones al texto, comúnmente para eliminar el subrayado de los enlaces o añadirlo a otros elementos. Los valores incluyen none (sin decoración), underline (subrayado), overline (línea por encima) y line-through (tachado). El valor blink está obsoleto y no se recomienda usar. 

a {
    text-decoration: none; /* Elimina el subrayado de los enlaces */
}

.ejemplo-tachado {
    text-decoration: line-through;
    color: red; /* Se pueden combinar propiedades */
}

text-align

Define la alineación horizontal del contenido en línea (como texto e imágenes) dentro de un elemento de bloque. Los valores posibles son left, right, center y justify. 

.parrafo-centrado {
    text-align: center;
}

.parrafo-justificado {
    text-align: justify;
}

text-indent

Especifica la sangría de la primera línea de un bloque de texto. Su valor se define con una unidad de medida (como px, em, %) o un valor relativo.

p {
    text-indent: 30px; /* La primera línea del párrafo tendrá una sangría de 30px */
}

Color

La propiedad color establece el color del texto de un elemento.

Los colores pueden especificarse de varias formas:

  • Nombres de color: Usando nombres predefinidos como red, blue, black, white, etc. Hay una amplia lista de nombres de color estándar.
  • Valores Hexadecimales: Usando el formato #RRGGBB o #RGB, donde RR, GG, BB son valores hexadecimales para Rojo, Verde y Azul (00 a FF).
  • Valores RGB/RGBA: Usando el formato rgb(rojo, verde, azul) o rgba(rojo, verde, azul, alfa), donde rojo, verde, azul son valores numéricos (0-255 o 0%-100%) y alfa es la opacidad (0 a 1).
  • Valores HSL/HSLA: Usando el formato hsl(tono, saturacion, luminosidad) o hsla(tono, saturacion, luminosidad, alfa).

Ejemplos de Propiedad color

.texto-rojo-nombre {
    color: red; /* Usando nombre de color */
}

.texto-rojo-hex {
    color: #FF0000; /* Usando valor hexadecimal */
}

.texto-rojo-rgb {
    color: rgb(255, 0, 0); /* Usando valor RGB */
}

.texto-semitransparente {
    color: rgba(0, 0, 0, 0.5); /* Texto negro con 50% de opacidad */
}

Fondo (Background)

Las propiedades de fondo controlan el color o la imagen que aparece detrás del contenido de un elemento.

background-color

Establece el color de fondo de un elemento. El valor se especifica de la misma manera que el color de texto (nombres, hex, RGB, etc.).

body {
    background-color: #f0f0f0; /* Color de fondo para toda la página */
}

.caja-con-fondo {
    background-color: lightblue;
}

background-image

Establece una imagen como fondo de un elemento. Se utiliza la función url() para especificar la ruta de la imagen.

body {
    background-image: url("imagenes/fondo-pagina.jpg");
}

.elemento-con-icono {
    background-image: url("../iconos/check.png");
}

La ruta puede ser absoluta (/imagenes/fondo.jpg) o relativa (../iconos/check.png).

Propiedad abreviada background

Esta propiedad es una abreviatura para varias propiedades de fondo (background-color, background-image, background-repeat, background-position, background-size, etc.). El orden no es estrictamente fijo para todos los valores, pero es común especificar primero la imagen y luego el color de fondo. 

body {
    /* Imagen de fondo (sin repetir), centrada, con color de fondo sólido */
    background: url("imagenes/fondo-pagina.jpg") no-repeat center center / cover, #f0f0f0;
}

.caja-resumida {
    /* Imagen y color de fondo */
    background: url("graficos/patron.png") white;
}

Es recomendable definir un background-color junto con un background-image para asegurar que haya un fondo visible si la imagen no se carga.

Espaciado (Margin y Padding)

Estas propiedades controlan el espacio alrededor del contenido de un elemento.

  • padding: Crea espacio *dentro* del borde del elemento, entre el contenido y el borde.
  • margin: Crea espacio *fuera* del borde del elemento, separándolo de otros elementos adyacentes.

padding (Relleno Interior)

Controla el espacio entre el contenido y el borde de un elemento. Puede definirse para los cuatro lados (arriba, derecha, abajo, izquierda) individualmente o usando la propiedad abreviada.

/* Propiedades individuales */
.caja-padding {
    padding-top: 10px;
    padding-right: 15px;
    padding-bottom: 10px;
    padding-left: 15px;
}

/* Propiedad abreviada */
.caja-padding-shorthand {
    padding: 10px 15px; /* Arriba/Abajo: 10px, Izquierda/Derecha: 15px */
}

.caja-padding-todos-lados {
    padding: 20px; /* 20px en los 4 lados */
}

Con la abreviada, si se dan 4 valores: Arriba Derecha Abajo Izquierda. Si se dan 2 valores: Arriba/Abajo Izquierda/Derecha. Si se da 1 valor: Los 4 lados.

margin (Margen Exterior)

Controla el espacio alrededor de un elemento, separándolo de otros. Se define de manera similar al padding, individualmente o usando la propiedad abreviada. 

/* Propiedades individuales */
.elemento-margin {
    margin-top: 20px;
    margin-bottom: 20px;
    margin-left: auto; /* Centra el elemento si tiene un ancho definido */
    margin-right: auto;
}

/* Propiedad abreviada */
.elemento-margin-shorthand {
    margin: 1em 0; /* Arriba/Abajo: 1em, Izquierda/Derecha: 0 */
}

Los márgenes verticales de elementos adyacentes pueden colapsar. Usar margin: auto; en los lados izquierdo y derecho es una técnica común para centrar un elemento de bloque dentro de su contenedor, siempre que el elemento tenga un width definido.

Dimensiones

width (Ancho)

Define el ancho del área de contenido de un elemento. Se puede especificar con unidades de longitud (px, em, rem, vw, etc.) o con porcentajes (%), que son relativos al ancho del contenedor padre. El valor auto (por defecto) permite que el navegador calcule el ancho.

.caja-ancho-fijo {
    width: 300px;
}

.caja-ancho-relativo {
    width: 80%; /* Ocupa el 80% del ancho del contenedor padre */
}

.imagen-ancho-maximo {
    max-width: 100%; /* La imagen nunca será más ancha que su contenedor */
    height: auto; /* Mantiene la proporción original */
}

También existen las propiedades min-width y max-width para controlar el rango de anchos posibles, lo cual es fundamental para el diseño web responsivo.

height (Alto)

Define el alto del área de contenido de un elemento. Se especifica de forma similar a width, usando unidades de longitud (px, em, rem, vh, etc.) o porcentajes (%). El valor auto (por defecto) permite que el navegador calcule el alto basado en el contenido.

.caja-alto-fijo {
    height: 150px;
}

.seccion-pantalla-completa {
    height: 100vh; /* Ocupa el 100% del alto de la ventana (viewport) */
}

De manera similar a width, existen min-height y max-height para controlar el rango de alturas posibles.

Borde (Border)

Las propiedades de borde permiten definir el grosor, estilo y color del borde alrededor de un elemento.

border-width (Grosor del Borde)

Define el grosor del borde. Se puede especificar con unidades de longitud (px, em, etc.) o con palabras clave predefinidas como thin, medium, thick.

/* Aplicar el mismo grosor a los 4 lados */
.caja-borde-ancho {
    border-width: 2px;
}

/* Grosor diferente para cada lado (Arriba Derecha Abajo Izquierda) */
.caja-bordes-varios {
    border-width: 1px 2px 3px 4px;
}

/* Grosor solo para un lado */
.caja-borde-superior {
    border-top-width: 5px;
}

border-style (Estilo del Borde)

Define el estilo de la línea del borde. Esta propiedad es fundamental; un borde no será visible a menos que se defina un estilo (incluso si se definen grosor y color). Valores comunes incluyen:

  • none: Sin borde (valor por defecto).
  • solid: Una línea recta sólida.
  • dashed: Una línea de guiones.
  • dotted: Una línea de puntos.
  • double: Un borde doble.
  • groove, ridge, inset, outset: Efectos 3D (dependen del color).
.caja-borde-solido {
    border-style: solid;
}

.caja-borde-punteado {
    border-style: dotted;
}

Al igual que con border-width, existen propiedades para estilos individuales por lado: border-top-style, border-right-style, etc.

border-color (Color del Borde)

Define el color del borde. Los valores se especifican de la misma manera que el color de texto o fondo (nombres, hex, RGB, etc.).

.caja-borde-azul {
    border-color: blue;
}

.caja-bordes-colores {
     /* Color diferente para cada lado (Arriba Derecha Abajo Izquierda) */
    border-color: red green blue yellow;
}

Propiedades individuales por lado: border-top-color, border-right-color, etc.

Propiedad abreviada border

La propiedad abreviada border es la forma más común y concisa de definir un borde. Combina border-width, border-style y border-color en una sola declaración. El orden recomendado es: [border-width] [border-style] [border-color]. 

.caja-borde-completo {
    border: 1px solid black; /* Grosor, Estilo, Color */
    padding: 10px; /* Añadimos padding para separar el contenido del borde */
}

.caja-sin-borde-abajo {
    border: 2px solid red; /* Borde en todos los lados por defecto */
    border-bottom: none; /* Sobrescribimos el borde inferior */
}

Usar la abreviada es más eficiente. Recuerda que el border-style es obligatorio para que el borde sea visible.

Posicionamiento y Visualización de Elementos

Además de controlar el aspecto de los elementos (color, tamaño, texto), CSS es fundamental para definir cómo se muestran y dónde se colocan en la página. Las propiedades display, float y position, junto con las propiedades de desplazamiento, son clave para controlar el layout y la disposición de los elementos.

display

Esta propiedad fundamental define el tipo de caja de renderizado que genera un elemento. Controla cómo el elemento se comporta en el flujo normal del documento y cómo interactúa con otros elementos.

Los valores más comunes son:

  • block: El elemento genera una caja de bloque. Ocupa todo el ancho disponible y siempre comienza en una nueva línea. Ejemplos: <div>, <p>, <h1>.
  • inline: El elemento genera una o más cajas en línea. No comienza en una nueva línea y solo ocupa el ancho necesario para su contenido. No se pueden controlar width, height, margin-top, ni margin-bottom. Ejemplos: <span>, <a>, <strong>.
  • inline-block: El elemento se comporta como un elemento en línea en relación con sus elementos hermanos (no comienza en una nueva línea), pero dentro de sí mismo se comporta como un bloque, permitiendo controlar width, height y los márgenes verticales/horizontales.
  • none: El elemento y su contenido no se muestran en absoluto y no ocupan espacio en el layout.
  • flex, grid: Valores que convierten el contenedor en un Flex Container o Grid Container, respectivamente, habilitando potentes modelos de layout modernos.
.elemento-bloque {
    display: block;
}

.elemento-inline {
    display: inline;
}

.elemento-inline-bloque {
    display: inline-block;
    width: 150px;
    height: 100px;
    margin: 10px;
}

.elemento-oculto {
    display: none;
}

float

La propiedad float posiciona un elemento (generalmente una imagen o un bloque de texto) a la izquierda o derecha dentro de su contenedor, permitiendo que el contenido en línea (como texto) fluya alrededor de él. Originalmente usado para maquetación básica, hoy en día Flexbox y Grid son a menudo preferidos para layouts complejos.

  • left: Flota el elemento a la izquierda.
  • right: Flota el elemento a la derecha.
  • none: El elemento no flota (valor por defecto).
.imagen-flotante {
    float: right; /* La imagen se moverá a la derecha */
    margin-left: 15px; /* Añadir espacio a su izquierda */
}

Si utilizas float, a menudo necesitarás usar la propiedad clear en un elemento posterior para detener el flujo de contenido y evitar que se solape con el elemento flotante.

position

Esta propiedad define el método de posicionamiento de un elemento en el documento. Funciona conjuntamente con las propiedades top, right, bottom y left.

  • static: Es el valor por defecto. El elemento se posiciona según el flujo normal del documento. Las propiedades top, right, bottom, left y z-index no tienen efecto.
  • relative: El elemento se posiciona según el flujo normal, pero su posición final puede ser ajustada *relativa a su posición original* utilizando top, right, bottom, left. El espacio que ocupaba en el flujo normal se mantiene.
  • absolute: El elemento se retira del flujo normal del documento y se posiciona *relativo a su ancestro posicionado más cercano* (un ancestro con position distinto de static). Si no tiene ancestros posicionados, se posiciona relativo al viewport inicial. Las propiedades top, right, bottom, left definen su desplazamiento desde los bordes de referencia.
  • fixed: El elemento se retira del flujo normal y se posiciona *relativo al viewport*. Permanece en su posición incluso cuando la página se desplaza. Es útil para crear encabezados o pies de página fijos. Las propiedades top, right, bottom, left definen su desplazamiento desde los bordes del viewport.
  • sticky: El elemento se posiciona según el flujo normal hasta que su contenedor se desplaza hasta un umbral especificado (usando top, right, bottom, o left), momento en el que se "pega" al viewport o a su contenedor, comportándose de forma similar a fixed. 
/* Un contenedor para demostrar position: absolute */
.contenedor-relativo {
    position: relative; /* Este es el ancestro posicionado */
    width: 300px;
    height: 200px;
    border: 1px solid blue;
    padding: 10px;
}

/* Elementos con distintos tipos de posicionamiento */
.elemento-estatico {
    /* position: static; es el valor por defecto, no suele declararse explícitamente */
    background-color: lightgray;
}

.elemento-relativo {
    position: relative;
    top: 10px; /* Se mueve 10px hacia abajo desde su posición normal */
    left: 10px; /* Se mueve 10px hacia la derecha desde su posición normal */
    background-color: lightgreen;
}

.elemento-absoluto {
    position: absolute;
    top: 20px; /* 20px desde el borde superior del contenedor relativo */
    right: 20px; /* 20px desde el borde derecho del contenedor relativo */
    background-color: lightcoral;
}

.elemento-fijo {
    position: fixed;
    bottom: 10px; /* 10px desde el borde inferior del viewport */
    right: 10px; /* 10px desde el borde derecho del viewport */
    background-color: yellow;
    padding: 5px;
}

/* Ejemplo conceptual de sticky (requiere scroll para verse el efecto completo) */
.elemento-sticky {
    position: sticky;
    top: 0; /* Se pegará en la parte superior del viewport cuando se desplace */
    background-color: lightblue;
    padding: 10px;
    margin-bottom: 15px;
}

La propiedad z-index (no cubierta aquí) se utiliza para controlar el orden de apilamiento de los elementos posicionados (quién aparece encima de quién).

top, right, bottom, left

Estas cuatro propiedades se utilizan para especificar el desplazamiento final de un elemento que tiene una propiedad position con valor relative, absolute, fixed, o sticky.

  • top: Desplaza desde el borde superior de referencia.
  • right: Desplaza desde el borde derecho de referencia.
  • bottom: Desplaza desde el borde inferior de referencia.
  • left: Desplaza desde el borde izquierdo de referencia.

El "borde de referencia" depende del valor de position:

  • Para position: relative;, el borde de referencia es la posición original del elemento.
  • Para position: absolute;, el borde de referencia es el padding edge del ancestro posicionado más cercano (o del viewport inicial).
  • Para position: fixed;, el borde de referencia es el viewport.
  • Para position: sticky;, actúan como umbral de desplazamiento dentro del contenedor o viewport.
.caja-posicionada {
    position: absolute; /* O relative, fixed, sticky */
    top: 50px; /* Desplazado 50px desde arriba */
    left: 100px; /* Desplazado 100px desde la izquierda */
    width: 100px;
    height: 100px;
    background-color: plum;
}
