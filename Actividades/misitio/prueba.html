Actividad: Sitio Web con HTML y CSS

En esta actividad vas a crear un sitio web compuesto por tres páginas enlazadas mediante un menú. Practicarás el uso de etiquetas básicas de HTML y aplicarás estilos con una hoja CSS externa.

  • p: Párrafos de texto.
  • div: Contenedores de contenido.
  • table: Tablas de datos.
  • a: Enlaces entre páginas.
  • img: Imágenes.
  • ul, ol, li: Listas ordenadas y desordenadas.
  • span: Elementos en línea.
  • em: Cursiva.
  • strong: Negrita.

¿Qué debes hacer?

Página 1 (index.html):

  • Crea un menú de navegación con enlaces a las tres páginas.
  • Añade un título, un párrafo de bienvenida y un <div> con listas ordenadas y desordenadas.
  • Incluye enlaces, texto en cursiva y en negrita.

Página 2 (pagina2.html):

  • Incluye un título, un párrafo y una galería de al menos tres imágenes.
  • Organiza las imágenes en un <div class="galeria">.

Página 3 (pagina3.html):

  • Incluye un título, un párrafo y una tabla con al menos 3 columnas y 2 filas.

CSS (estilos.css):

  • Crea una hoja de estilos externa y enlázala en todas las páginas.
  • Aplica estilo al menú, listas, imágenes, tablas, etc.
