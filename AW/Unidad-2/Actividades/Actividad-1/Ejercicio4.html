Ejercicio 3: Comprobar Edad

En esta actividad, el objetivo principal es manipular el DOM para cambiar dinámicamente el color de un elemento de texto utilizando JavaScript. Se busca que el usuario interactúe con botones que, al ser pulsados, modifiquen la apariencia de un párrafo específico.

Puedes añadir un unico boton y pasarle un color en ingles para que lo aplique al párrafo.

Deberás desarrollar la lógica necesaria para:

  • Identificación de Elementos: Seleccionar un elemento HTML (un párrafo en este caso) por su ID para poder modificar sus propiedades. Puedes añadir más elementos para modificar su color.
  • Manejo de Eventos: Asociar una función JavaScript a eventos de clic en múltiples botones. Cada botón debe invocar la misma función, pero pasándole un argumento diferente (el color deseado).
  • Manipulación de Estilos: Dentro de la función, acceder a la propiedad `style.color` del elemento seleccionado y asignarle el valor del color recibido como argumento.
  • Interactividad: Demostrar cómo diferentes interacciones del usuario (clics en distintos botones) pueden llevar a cambios visuales inmediatos en la página web.

El código HTML es el siguiente:


<!DOCTYPE html>
<html lang="es">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ejercicio 4: Cambiar Color</title>
</head>

<body>

    <h1>Ejercicio 4: Cambiar Color de Texto</h1>

    <p id="textoEjemplo">Este es un párrafo de ejemplo. Haz clic en los botones para cambiar su color.</p>

    <!-- Cada botón llama a la misma función, pero le pasa un color diferente como argumento -->
    <button onclick="cambiarColor('red')">Rojo</button>
    <button onclick="cambiarColor('green')">Verde</button>
    <button onclick="cambiarColor('blue')">Azul</button>

    <script>
        // La función acepta un parámetro, 'colorSeleccionado', que cambiará
        // dependiendo del botón que se pulse.
        function cambiarColor(colorSeleccionado) {
            // 1. Seleccionamos el párrafo por su ID
            const parrafo = document.getElementById("textoEjemplo");

            // 2. Accedemos a sus estilos (style) y cambiamos la propiedad 'color'
            parrafo.style.color = colorSeleccionado;
        }
    </script>
</body>

</html>
