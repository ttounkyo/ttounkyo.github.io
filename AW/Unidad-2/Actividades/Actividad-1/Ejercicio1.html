        Púlsame
    </button>

    <h2>¿Cómo funciona?</h2>
    <ul>
        <li><button>: Es la etiqueta estándar de HTML para crear un botón.</li>
        <li>onclick="...": Es un "atributo de evento". Le dice al navegador: "Cuando alguien haga clic en este elemento,
            ejecuta el código que está entre las comillas".</li>
        <li><alert('...')>: Es una función de JavaScript que muestra una ventana emergente (un pop-up) con el
            mensaje que le pases entre paréntesis.</li>
    </ul>
