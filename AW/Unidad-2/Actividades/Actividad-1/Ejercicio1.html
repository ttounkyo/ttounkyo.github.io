Ejercicio 1: Botón con Alerta

En este ejercicio se muestra un botón que, al hacer clic en él, muestra una alerta con un mensaje personalizado.

Ahora modificalo tal y como hemos realizado en clase, con otros elementos como prompt, confirm y createElement

El código HTML es el siguiente:


<!DOCTYPE html>
<html lang="es">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ejercicio 1: Botón con Alerta</title>
</head>

<body>

    <h1>Ejercicio 1: Botón con Alerta</h1>

    <!--
        El atributo 'onclick' ejecuta el código JavaScript que contiene
        cuando el usuario hace clic en este botón.
    -->
    <button onclick="alert('Bienvenido/a a mi página web')">
        Púlsame
    </button>

    <h2>¿Cómo funciona?</h2>
    <ul>
        <li><button>: Es la etiqueta estándar de HTML para crear un botón.</li>
        <li>onclick="...": Es un "atributo de evento". Le dice al navegador: "Cuando alguien haga clic en este elemento,
            ejecuta el código que está entre las comillas".</li>
        <li><alert('...')>: Es una función de JavaScript que muestra una ventana emergente (un pop-up) con el
            mensaje que le pases entre paréntesis.</li>
    </ul>
    <!-- Lo que hicimos en clase sobre el ejercicio 1-->
    <button onclick="alerta()">Click me</button>
    <script>
        //Función alerta()
        function alerta() {
            //Pedimos el nombre al usuario
            let nombre = prompt("Introduce tu nombre");
            //Si el usuario no introduce ningún nombre
            if (nombre !== "") {
                //Pedimos confirmación
                let ok = confirm(`¿Estás seguro de que eres ${nombre}?`);
                //Si el usuario confirma
                if (ok) {
                    //Creamos un párrafo
                    let p = document.createElement("p").append(`Bienvenido/a ${nombre} a mi página web`);
                    //Lo añadimos al body
                    document.body.appendChild(p);
                } else {
                    //Si el usuario no confirma
                    alert("No has confirmado que eres " + nombre + ", repite el proceso");
                    //Repetimos la función
                    alerta();
                }
            } else {
                //Si el usuario no introduce ningún nombre
                alert("No has introducido ningún nombre, repite el proceso");
                //Repetimos la función
                alerta();
            }
        }
    </script>
</body>

</html>
