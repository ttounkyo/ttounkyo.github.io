Introducción a JavaScript

Este capítulo presenta el lenguaje de programación JavaScript y analiza algunos de sus conceptos fundamentales, desde su propósito y su historia hasta cómo empezar a escribir tus primeras líneas de código.

¿Qué Debes Conocer Previamente?

Esta guía presume que tienes los siguientes antecedentes básicos:

  • Una comprensión general de Internet y la World Wide Web.
  • Un buen conocimiento práctico del Lenguaje de Marcado de Hipertexto (HTML).
  • Alguna experiencia en programación. Si eres completamente nuevo, te recomendamos empezar con un tutorial básico de lógica de programación.

¿Qué es JavaScript?

JavaScript es un lenguaje de programación multiplataforma y orientado a objetos que se utiliza para hacer que las páginas web sean interactivas. Gracias a él, podemos crear desde animaciones complejas y menús desplegables hasta aplicaciones web completas que se actualizan en tiempo real.

JavaScript contiene una biblioteca estándar de objetos, como Array, Date y Math, y un conjunto básico de elementos del lenguaje como operadores, estructuras de control y declaraciones. El núcleo de JavaScript se puede extender para una variedad de propósitos completándolo con objetos adicionales:

  • JavaScript de lado del cliente (Client-Side): Es el JavaScript que se ejecuta en el navegador web. Se extiende proporcionando objetos para controlar el navegador y su Modelo de Objetos de Documento (DOM). Permite que una aplicación coloque elementos en un formulario HTML, responda a clics del ratón, valide datos y cambie dinámicamente el contenido de la página sin necesidad de recargarla.
  • JavaScript de lado del servidor (Server-Side): Versiones más avanzadas como Node.js amplían el núcleo del lenguaje con objetos relevantes para ejecutar JavaScript en un servidor. Esto permite a una aplicación comunicarse con una base de datos, gestionar usuarios, manipular archivos en el servidor y construir APIs completas.

¡No Confundir! JavaScript y Java

JavaScript y Java son similares en algunos aspectos superficiales, pero fundamentalmente diferentes. El lenguaje JavaScript se parece a Java en la sintaxis, pero no tiene su tipado estático ni su fuerte verificación de tipos. La similitud en el nombre es principalmente una decisión de marketing de la época.

JavaScript Java
Orientado a objetos basado en prototipos. La herencia es dinámica y las propiedades y métodos se pueden agregar a cualquier objeto en tiempo de ejecución. Basado en clases. Los objetos son instancias de clases y la herencia se define en la jerarquía de clases. No se pueden añadir propiedades dinámicamente.
Los tipos de datos de las variables no se declaran (tipado dinámico). Los tipos de datos de las variables deben declararse (tipado estático).
Se ejecuta principalmente en el navegador o en entornos como Node.js. Por seguridad, no puede acceder directamente al sistema de archivos del cliente. Puede ejecutarse en una máquina virtual (JVM) en cualquier sistema y tiene acceso completo a los recursos del sistema, como el disco duro.

JavaScript y la Especificación ECMAScript

JavaScript está estandarizado por Ecma International. Esta versión estandarizada se denomina ECMAScript y su objetivo es asegurar que el lenguaje se comporte de la misma manera en todas las plataformas que lo soporten (navegadores, servidores, etc.). El estándar ECMAScript está documentado en la especificación ECMA-262.

Documentación vs. Especificación

La especificación ECMA-262 es un documento técnico para los ingenieros que crean los motores de JavaScript (como V8 en Chrome o SpiderMonkey en Firefox). No está pensada para aprender a programar.

Para aprender a escribir código, debes utilizar la documentación de JavaScript, como la de MDN Web Docs, que describe el lenguaje desde la perspectiva de un programador.

Cómo Empezar con JavaScript

Comenzar con JavaScript es fácil: todo lo que necesitas es un navegador web moderno. La Consola Web integrada en los navegadores es una herramienta increíblemente útil para experimentar.

Para abrir la Consola Web, presiona Ctrl+Shift+J (o F12) en Windows/Linux, o Cmd+Option+J en Mac, y selecciona la pestaña "Consola".

La Consola Web es tu primer campo de pruebas para JavaScript.

Hola Mundo en la Consola

Para comenzar a escribir JavaScript, abre la Consola Web y escribe tu primer código "Hola mundo":


(function () {
  "use strict";
  /* Inicio de tu código */

  function greetMe(yourName) {
    alert("Hola " + yourName);
  }

  greetMe("Mundo");

  /* Fin de tu código */
})();

Presiona Ctrl+Enter (o haz clic en el botón Ejecutar si usas el modo multilínea) para ver el resultado.

Una Buena Práctica: "use strict" y las IIFE

Por ahora, acostúmbrate a envolver tu código de esta manera. La estructura (function() { ... })(); se llama IIFE (Immediately Invoked Function Expression) y, junto con "use strict";, te ayudará a:

  • Evitar errores comunes que confunden a los principiantes.
  • Prevenir que las variables de tu código interfieran con otras partes de la página web.
