1. Conceptos Básicos sobre Aplicaciones Ofimáticas

Para entender las aplicaciones ofimáticas, es fundamental conocer dos términos clave:

  • Informática: Es la ciencia que estudia el tratamiento automático de la información a través de computadoras y sistemas informáticos.
  • Ofimática: Es el conjunto de técnicas, aplicaciones y herramientas informáticas que se utilizan para optimizar, automatizar y mejorar las tareas y funciones relacionadas con la oficina. Su objetivo es facilitar el trabajo de oficina y gestionar la información de manera más eficiente.

1.1. Tipos de Aplicaciones Ofimáticas

Las suites ofimáticas integran diversas herramientas, cada una diseñada para una función específica:

  • Procesadores de texto: Permiten crear, editar, dar formato e imprimir documentos basados en texto, como cartas, informes o libros. Ejemplos: Microsoft Word, Writer de LibreOffice, Google Docs.
  • Hojas de cálculo: Son programas que organizan datos en filas y columnas, permitiendo realizar cálculos complejos, análisis de datos y crear gráficos. Ejemplos: Microsoft Excel, Calc de LibreOffice, Google Sheets.
  • Gestores de bases de datos: Herramientas para almacenar, organizar, consultar y gestionar grandes volúmenes de información de manera estructurada. Ejemplos: Microsoft Access, Base de LibreOffice.
  • Programas de presentación: Sirven para crear diapositivas con texto, imágenes, gráficos y multimedia, ideales para exposiciones y conferencias. Ejemplos: Microsoft PowerPoint, Impress de LibreOffice, Google Slides.
  • Gestores de agenda y correo electrónico: Facilitan la comunicación, la organización de contactos, la gestión de calendarios y la administración de correos electrónicos. Ejemplos: Microsoft Outlook, Mozilla Thunderbird, Gmail.
  • Editores de imagen y vídeo: Aunque no siempre se consideran parte central de una suite ofimática tradicional, cada vez más se integran herramientas básicas para editar o insertar imágenes y y vídeos en documentos y presentaciones. Ejemplos básicos: Paint, herramientas de recorte. (Programas avanzados serían Photoshop, GIMP, etc.).

1.2. Licencias de Software

Las licencias de software definen los términos legales bajo los cuales se puede usar, distribuir, modificar y copiar un programa informático.

  • Definición de licencias software: Contrato legal entre el autor (o titular de los derechos de autor) y el usuario final, estableciendo los permisos y restricciones para el uso del software.
  • Software propietario: Es aquel software cuyo código fuente no está disponible y sobre el cual el usuario tiene restricciones para modificar, redistribuir o incluso, en algunos casos, usarlo. Su uso suele requerir el pago de una licencia. Ejemplos: Microsoft Office, Adobe Photoshop.
  • Software libre: Es aquel que respeta la libertad de los usuarios y la comunidad. Otorga a los usuarios la libertad de ejecutar, copiar, distribuir, estudiar, cambiar y mejorar el software. Las "cuatro libertades esenciales" son clave. Ejemplos: GNU/Linux, LibreOffice, GIMP.
  • Copyright, dominio público y copyleft:
    • Copyright: Es un conjunto de derechos legales exclusivos otorgados al creador de una obra original (incluido el software) para su publicación, distribución y adaptación. Simbolizado por ©.
    • Dominio público: Obras cuyos derechos de autor han expirado o nunca existieron, lo que significa que pueden ser utilizadas por cualquier persona sin restricciones de copyright.
    • Copyleft: Es un tipo de licencia de software libre que, utilizando la legislación de derechos de autor, garantiza que el software y sus versiones modificadas permanezcan libres, es decir, que nadie pueda apropiarse de las modificaciones realizadas y distribuirlas como software propietario. Un ejemplo es la licencia GPL (General Public License).
  • Software comercial, shareware y freeware:
    • Software comercial: Software que se distribuye con fines de lucro. Puede ser propietario o, en algunos casos, libre (ej. una versión premium de un software libre).
    • Shareware: Software que se distribuye de forma gratuita por un período de prueba limitado o con funcionalidades restringidas. Una vez finalizado el período o para desbloquear todas las funciones, se debe adquirir la licencia.
    • Freeware: Software que se distribuye de forma gratuita para su uso, sin coste alguno. A diferencia del software libre, el freeware no garantiza acceso al código fuente ni libertad para modificarlo o redistribuirlo.

1.3. Suites de Aplicaciones Ofimáticas

Una suite ofimática es un paquete de aplicaciones que integran las herramientas esenciales para las tareas de oficina, compartiendo a menudo una interfaz y funcionalidades comunes.

Podéis encontrar más información aquí https://es.wikipedia.org/wiki/Paquete_de_oficina

Suites Ofimáticas Locales vs. Suites Ofimáticas en la Nube

Actualmente, existen gran cantidad de suites ofimáticas locales (es decir, se ejecutan y tratan los datos en nuestro ordenador) y suites ofimáticas en la nube (se ejecutan en la nube, que básicamente es el ordenador de otra persona).

La nube es realmente “el ordenador de otra persona”. Hay que tener cuidado con lo que se usa en la nube, ya que incluso aunque una aplicación en la nube prometa que no almacenará nada, no tenemos garantía de que así sea y lo usen para algún fin malicioso.

Hay que tener muy claro que todo lo que se trabaja en la nube tiene una serie de peligros, además de los mencionados anteriormente. Aquí un enlace con más información al respecto: https://blog.netdatanetworks.com/5-riesgos-en-la-nube-y-como-gestionarlos

  • Suites Ofimáticas Locales:
    • Microsoft Office: Es la suite ofimática propietaria más popular y utilizada a nivel mundial, desarrollada por Microsoft. Incluye programas como Word, Excel, PowerPoint, Outlook y Access, que se instalan directamente en el equipo del usuario.
    • OpenOffice.org: Fue una suite ofimática de código abierto y software libre. Ha sido la base para otras suites libres y ofrecía alternativas a los programas de Microsoft Office.
    • LibreOffice: Es una suite ofimática de software libre y código abierto, bifurcada de OpenOffice.org. Es ampliamente utilizada y mantenida por The Document Foundation, ofreciendo una alternativa robusta y compatible a Microsoft Office. Incluye Writer, Calc, Impress, Draw, Base y Math.
  • Suites Ofimáticas en la Nube:
    • Google Workspace (anteriormente G Suite): Ofrecida por Google, incluye Google Docs (procesador de texto), Google Sheets (hoja de cálculo), Google Slides (presentaciones), Google Drive (almacenamiento) y Gmail (correo electrónico). Permite la colaboración en tiempo real y el acceso desde cualquier dispositivo con conexión a internet.
    • Microsoft 365 (anteriormente Office 365): Es la versión en la nube de Microsoft Office. Ofrece acceso a las aplicaciones clásicas (Word, Excel, PowerPoint) vía web o mediante instalación local con licencia de suscripción, además de almacenamiento en OneDrive y servicios de colaboración.
    • OnlyOffice: Una suite ofimática online y de código abierto que ofrece procesador de texto, hoja de cálculo y presentaciones, con fuerte enfoque en la compatibilidad con formatos de Microsoft Office y herramientas de colaboración.

Nosotros trabajaremos en la Suite de Google, ya que está disponible en el centro y los documentos en los que trabajamos no contienen información de caracter personal.

Evitad la nube cuando tratéis con datos personales. Recordad, además de problemas de robo de datos, puede haber problemas legales. La legislación española, por ejemplo, prohíbe almacenar datos personales de terceros en servidores que estén fuera de la Unión Europea, por lo cual almacenarlos en algo como Google Drive, salvo que se pague para que Google use un servidor Europeo, podría ser delito. Hay que cerciorarse, incluso para contextos de mucha confianza, si es legal que almacenemos según qué datos, en según qué servidores, de según qué aplicaciones.

2. La Suite Google Workspace (Google Docs, Sheets, Slides)

Google Workspace es una de las suites ofimáticas en la nube más populares, destacando por su facilidad de uso, colaboración en tiempo real y accesibilidad desde cualquier dispositivo. A diferencia de las suites locales, Google Workspace reside completamente en la nube, lo que implica consideraciones específicas.

2.1. Acceso y Uso

  • Basado en la web: No requiere instalación de software en el equipo. Se accede a través de un navegador web desde cualquier dispositivo con conexión a Internet.
  • Almacenamiento en la nube: Todos los documentos se guardan automáticamente en Google Drive. Esto facilita el acceso desde diferentes ubicaciones y la sincronización entre dispositivos.
  • Colaboración en tiempo real: Permite a múltiples usuarios trabajar simultáneamente en el mismo documento, viendo los cambios en tiempo real y comunicándose a través de comentarios o chats integrados.
  • Compatibilidad de formatos: Aunque usa formatos nativos de Google, es compatible con la importación y exportación de documentos de Microsoft Office (.docx, .xlsx, .pptx) y otros.

2.2. Ventajas y Desventajas de las Suites en la Nube (Google Workspace)

Ventajas:

  • Accesibilidad: Se puede acceder a los documentos desde cualquier lugar y dispositivo con conexión a Internet.
  • Colaboración: Facilita el trabajo en equipo con edición en tiempo real.
  • Ahorro de espacio: No consume espacio en disco local para el software ni para los documentos.
  • Copia de seguridad automática: Los documentos se guardan y versionan automáticamente en la nube.
  • Actualizaciones automáticas: El software se mantiene siempre actualizado sin intervención del usuario.

Desventajas y Riesgos:

  • Dependencia de Internet: Requiere una conexión a Internet activa para funcionar plenamente (aunque algunos ofrecen modo sin conexión limitado).
  • Seguridad y Privacidad de Datos: Los datos se almacenan en servidores de terceros, lo que genera preocupaciones sobre la privacidad y el control de la información, especialmente si se manejan datos sensibles o personales (ver atención en el punto 1.3).
  • Cumplimiento normativo: Es crucial verificar que el uso de estas suites cumpla con las leyes de protección de datos locales (ej. GDPR en la UE) respecto a la ubicación de los servidores y el tratamiento de la información.
  • Rendimiento: Puede verse afectado por la velocidad de la conexión a Internet o por la carga de los servidores del proveedor.
  • Funcionalidades: Aunque muy completas, algunas suites en la nube pueden tener menos funcionalidades avanzadas que sus contrapartes de escritorio.

2.3. Configuración y Actualizaciones (en la nube)

En el caso de las suites en la nube, el usuario tiene menos control directo sobre la instalación y actualización, pero sí sobre la configuración de las aplicaciones y los documentos.

  • Configuración: Se realiza a través de las opciones de la cuenta de Google (o la cuenta de la suite) y dentro de cada aplicación (idioma, zona horaria, permisos de acceso a documentos, plantillas predeterminadas).
  • Actualizaciones: Las actualizaciones del software son gestionadas por el proveedor del servicio (Google en este caso) de manera transparente para el usuario, asegurando que siempre se está utilizando la última versión.

3. Documentación y Soporte Técnico

Tanto la documentación como el soporte técnico son recursos esenciales para aprovechar al máximo las aplicaciones y resolver problemas.

3.1. Documentación

La documentación proporciona información detallada sobre el uso y las funcionalidades de las aplicaciones. Es recomendable consultar las fuentes oficiales del fabricante.

  • Ayuda de la aplicación: La mayoría de las aplicaciones incluyen una función de ayuda integrada, accesible directamente desde el programa. Para Google Workspace, puedes encontrar la ayuda oficial aquí: Centro de Ayuda de Google Workspace.
  • Manuales de uso y tutoriales en línea: Documentos más extensos que guían al usuario a través de todas las funcionalidades. Los sitios de soporte de Google ofrecen guías detalladas, por ejemplo:
    • Ayuda de Google Docs: https://support.google.com/docs/?hl=es
    • Ayuda de Google Sheets: https://support.google.com/docs/answer/9334460?hl=es&ref_topic=1382883
    • Ayuda de Google Slides: https://support.google.com/docs/answer/9280907?hl=es&ref_topic=1382883
  • Recursos adicionales en línea: Foros, blogs, videos tutoriales y wikis creados por la comunidad o los desarrolladores, que ofrecen una vasta cantidad de información y soluciones prácticas.

3.2. Soporte Técnico

Cuando la documentación no es suficiente, el soporte técnico ofrece asistencia personalizada. Para las suites en la nube como Google Workspace, el soporte suele estar integrado en la plataforma.

  • Soporte técnico del proveedor: Para usuarios de Google Workspace (especialmente cuentas empresariales o educativas), Google ofrece canales de soporte dedicados. Para usuarios individuales de las aplicaciones gratuitas, el soporte es principalmente a través de la ayuda en línea y foros comunitarios. Puedes acceder al soporte general de Google aquí: Centro de Ayuda de Google.
  • Canales de soporte: Los canales comunes incluyen chat en línea, correo electrónico, teléfono o sistemas de tickets, donde un técnico puede guiar al usuario paso a paso (disponibilidad según el tipo de cuenta y servicio contratado).
  • Comunidades de usuarios: Los foros y comunidades en línea son una excelente fuente de soporte técnico peer-to-peer, donde otros usuarios con experiencia pueden ofrecer soluciones. Un ejemplo es la comunidad de ayuda de Google: Comunidades de Ayuda de Google.

4. Ergonomía y Riesgos Laborales en el Uso de Pantallas

Trabajar con pantallas de visualización de datos (PVD), ya sean ordenadores de sobremesa, portátiles, tablets o móviles, implica ciertos riesgos para la salud a corto y largo plazo que deben ser mitigados mediante prácticas ergonómicas adecuadas y descansos periódicos.

4.1. Riesgos Asociados al Uso de Pantallas

El uso prolongado e incorrecto de las PVD puede derivar en diversos problemas de salud:

  • Fatiga visual: Síntomas como sequedad ocular, irritación, visión borrosa, dolores de cabeza y molestias en los ojos, causados por la fijación prolongada, el brillo de la pantalla y la falta de parpadeo.
  • Trastornos musculoesqueléticos: Dolores y molestias en cuello, hombros, espalda y muñecas (ej. síndrome del túnel carpiano) debido a posturas incorrectas, movimientos repetitivos y falta de soporte ergonómico.
  • Estrés y fatiga mental: La carga de trabajo, la presión y la sobreexposición a información pueden generar estrés, irritabilidad y dificultad de concentración.
  • Problemas circulatorios: La inactividad prolongada puede afectar la circulación sanguínea en las extremidades.

4.2. Medidas Ergonómicas y Preventivas

Es fundamental adoptar medidas para minimizar estos riesgos:

  • Postura correcta:
    • Espalda recta y apoyada en el respaldo de la silla.
    • Pies apoyados en el suelo o en un reposapiés.
    • Brazos relajados, antebrazos paralelos al suelo y muñecas rectas.
    • Pantalla a la distancia de un brazo, con el borde superior a la altura de los ojos o ligeramente por debajo.
  • Iluminación adecuada: Evitar reflejos en la pantalla y asegurar una iluminación ambiental equilibrada, ni muy brillante ni muy tenue.
  • Descansos activos: Realizar pausas breves cada 45-60 minutos. Levantarse, caminar unos minutos, estirar los músculos y realizar ejercicios oculares (mirar objetos lejanos, parpadear frecuentemente).
  • Equipo ergonómico: Utilizar sillas ajustables, teclados y ratones ergonómicos, y, si es necesario, soportes para monitores o portátiles.
  • Ajustes de pantalla: Configurar el brillo, contraste y tamaño de fuente de la pantalla para que sean cómodos para la vista.

Si en algún momento queréis saber más de este tema, en las guías de riesgos laborales, a los riesgos asociados con las pantallas se les llama técnicamente riesgos en “Pantallas de visualización de datos”.

5. Cheat Sheets (Hojas de Trucos)

Una “Cheat Sheet” o “hoja de trucos” es un recurso compacto, generalmente de una o dos páginas, que condensa información clave, comandos, atajos de teclado o sintaxis fundamental sobre el uso de una aplicación, herramienta, lenguaje de programación o cualquier sistema informático. Su objetivo principal es servir como referencia rápida para mejorar la eficiencia y ahorrar tiempo.

Es muy interesante tener “Cheat sheets” de los programas o herramientas que usemos a diario, ya que nos harán la faena más fácil, reducirán la necesidad de memorizar y nos permitirán ahorrar tiempo, especialmente cuando estamos aprendiendo o trabajando bajo presión.

6. Búsqueda de Información en Internet

La capacidad de encontrar información de manera eficiente en internet es una habilidad fundamental. Para ello, es importante dominar tanto los motores de búsqueda tradicionales como las nuevas herramientas basadas en Inteligencia Artificial.

6.1. Búsqueda en Google (y otros motores de búsqueda)

Google es un poderoso motor de búsqueda que permite afinar tus consultas para obtener resultados más precisos. Dominar sus comandos y operadores es esencial:

  • Frases exactas: Utiliza comillas dobles "texto exacto" para buscar la frase tal cual.
  • Excluir palabras: Añade un guion -palabra para excluir resultados que contengan esa palabra.
  • Buscar en un sitio específico: Usa site:dominio.com para limitar la búsqueda a una web en particular.
  • Tipos de archivo: Usa filetype:pdf o filetype:docx para encontrar documentos de un formato específico.
  • Rangos numéricos: Puedes buscar un rango de números con número..número (ej. coche 2000..2023).
  • Palabras clave obligatorias: Usa +palabra (o simplemente la palabra, ya que Google lo hace por defecto) para asegurar que una palabra esté en los resultados.
  • Buscar sinónimos: Usa ~palabra para incluir sinónimos en tu búsqueda.

Aquí enlazamos a algunas “Cheat sheets” que contienen los principales comandos de búsqueda del motor de Google:

  • Google Guide: Advanced Operator Reference (PDF)
  • Google Search Education: Goodies
  • Google for Educators: Search Tips & Tricks (PDF)

6.2. Búsqueda con Inteligencia Artificial

La aparición de modelos de lenguaje grandes (LLMs) como ChatGPT, Gemini o Copilot ha transformado la forma en que interactuamos con la información. Estas herramientas ofrecen una aproximación conversacional a la búsqueda:

  • Respuestas directas y contextualizadas: En lugar de una lista de enlaces, los asistentes de IA pueden generar resúmenes, explicaciones o incluso código directamente a partir de tu pregunta.
  • Procesamiento de lenguaje natural avanzado: Entienden el contexto y la intención de preguntas complejas, permitiendo consultas más naturales y menos dependientes de palabras clave específicas.
  • Generación de contenido: Además de buscar, pueden ayudarte a generar ideas, redactar textos, traducir o reescribir información.
  • Integración en buscadores: Algunos motores de búsqueda (como Google con Search Generative Experience o Microsoft Bing con Copilot) están integrando la IA para ofrecer resultados híbridos, combinando enlaces con resúmenes generados por IA.

Aunque muy potentes, es crucial recordar que las respuestas de la IA deben ser contrastadas, ya que pueden generar información incorrecta o desactualizada ("alucinaciones"). Son excelentes herramientas para iniciar una investigación, pero no siempre la fuente definitiva de la verdad.

7. Instalación y Desinstalación de Software Local: Ejemplo LibreOffice

Este apartado te guiará a través del proceso de instalación y desinstalación de una suite ofimática local como LibreOffice, que es un excelente ejemplo de software libre y multiplataforma.

7.1. Requisitos del Sistema

Antes de proceder con la instalación, es fundamental verificar que el equipo cumpla con los requisitos mínimos del sistema para garantizar el correcto funcionamiento de la aplicación.

  • Requisitos de Software:
    • Sistema operativo compatible:
      • Microsoft Windows: versiones 7, 8, 8.1, 10, 11 o Windows Server.
      • GNU/Linux: kernel 3.10 o superior, glibc v2.20 o superior.
      • macOS: versión 10.14 (Mojave) o superior.
    • Java Runtime Environment (JRE): se recomienda disponer de una versión actualizada de JRE para aprovechar ciertas funcionalidades avanzadas, aunque no es estrictamente necesario para la instalación básica.
  • Requisitos de Hardware:
    • CPU: Procesador Intel Pentium III o compatible (se recomienda Intel Core 2 Duo o superior).
    • Memoria RAM: mínimo 256 MB (se recomienda 512 MB o más para un rendimiento óptimo).
    • Espacio en disco: al menos 1.5 GB de espacio libre para la instalación completa.
    • Pantalla: resolución mínima de 1024x768 píxeles con 256 colores (se recomienda una resolución y profundidad de color superiores).

7.2. Proceso de Instalación de LibreOffice

A continuación, proporcionaremos una guía de instalación completa, desde la búsqueda de la aplicación hasta la apertura del programa, utilizando LibreOffice como ejemplo. Se utilizarán referencias a imágenes para facilitar la comprensión de todo el proceso.

Búsqueda de la Aplicación

El primer paso es localizar la fuente oficial de descarga de LibreOffice. Es crucial descargar el software desde su sitio web oficial para asegurar que sea una versión segura y genuina.

  • Abre tu navegador web (Chrome, Firefox, Edge, etc.).
  • En la barra de búsqueda de Google (o tu buscador preferido), escribe "descargar LibreOffice" y presiona Enter.
  • Busca el resultado que te dirija al sitio oficial de The Document Foundation, usualmente es.libreoffice.org o www.libreoffice.org.

Búsqueda en Google de " descargar LibreOffice" y selección del enlace oficial.

Descarga de la Aplicación

Una vez en el sitio web oficial, procederemos a descargar el instalador.

  • Haz clic en el botón grande de "Descargar ahora" o "Download LibreOffice" en la página principal.
  • La web debería detectar automáticamente tu sistema operativo. Si no es así, selecciona la opción correcta (Windows, macOS o Linux).
  • Haz clic en "Descargar" junto a la versión estable más reciente. Generalmente se descargará un archivo .msi para Windows, .dmg para macOS o .deb/.rpm para Linux.

Página de descarga de LibreOffice con la opción de Windows y el botón " Descargar" visible.

Instalación de la Aplicación

Con el archivo descargado, iniciaremos el proceso de instalación en tu equipo.

  • Localiza el archivo de instalación descargado (usualmente en tu carpeta "Descargas").
  • Haz doble clic en el archivo (ej. LibreOffice_[versión]_Win_x64.msi) para iniciar el asistente de instalación.
  • Aparecerá una ventana de bienvenida del asistente de instalación. Haz clic en "Siguiente".
  • Selecciona el tipo de instalación (típicamente "Típica" es suficiente para la mayoría de usuarios) y haz clic en "Siguiente".
  • Si deseas cambiar la ubicación de instalación o seleccionar componentes específicos, elige "Personalizada"; de lo contrario, haz clic en "Instalar".
  • El asistente mostrará una barra de progreso mientras se copian los archivos. Puede que se te pida permiso de administrador.

Selección de instalación " Típica".

Selección creación acceso directo" .

Finalización de la Instalación

Una vez que el proceso de copia de archivos ha terminado, el asistente te notificará que la instalación ha sido completada.

  • Haz clic en " Finalizar" o "Terminar" para cerrar el asistente de instalación.
  • LibreOffice ya estará instalado en tu equipo y listo para usarse.

Ventana de finalización de la instalación.

Apertura del Programa

Para empezar a usar LibreOffice, simplemente ábrelo como cualquier otra aplicación en tu sistema.

  • En Windows: Puedes encontrarlo en el menú " Inicio", buscando "LibreOffice" , o a través de un acceso directo en el escritorio (si se creó durante la instalación).
  • En macOS: Estará en la carpeta "Aplicaciones".
  • En Linux: Generalmente en el menú de aplicaciones bajo "Oficina" o "Productividad".
  • Al abrirlo, verás la pantalla de inicio de LibreOffice desde donde puedes crear un nuevo documento (Writer, Calc, Impress, etc.) o abrir uno existente.

Pantalla de inicio de LibreOffice lista para crear o abrir un documento.

Configuración de la Aplicación (Opcional)

Durante o después de la instalación, podrías querer realizar algunas configuraciones básicas.

  • Después de la instalación, al abrir LibreOffice por primera vez, puedes encontrar opciones para configurar el idioma de la interfaz, la revisión ortográfica, y las asociaciones de tipos de archivo (para que los documentos de Office se abran con LibreOffice por defecto).
  • Estas configuraciones también se pueden ajustar en cualquier momento desde " Herramientas" -> "Opciones" dentro de cualquiera de las aplicaciones de LibreOffice (Writer, Calc, etc.).

7.3. Desinstalación de Software Local (Ejemplo LibreOffice)

Para desinstalar una aplicación local como LibreOffice, debes utilizar las herramientas de gestión de programas de tu sistema operativo. El proceso general es el siguiente:

  • En Windows:
    1. Abre el "Panel de control" y ve a "Programas" -> "Programas y características".
    2. Alternativamente, en Windows 10/11, puedes ir a "Configuración" -> "Aplicaciones" -> "Aplicaciones y características".
    3. Busca "LibreOffice" en la lista de programas instalados.
    4. Haz clic derecho sobre "LibreOffice" y selecciona "Desinstalar" (o haz clic en el botón "Desinstalar" si estás en Configuración).
    5. Sigue las instrucciones del asistente de desinstalación para eliminar completamente la aplicación de tu sistema.
  • En macOS:
    1. Abre la carpeta "Aplicaciones".
    2. Arrastra la aplicación "LibreOffice.app" a la papelera.
    3. Vacía la papelera.
  • En GNU/Linux:
    1. El método varía según la distribución (ej. Synaptic, Centro de Software, sudo apt remove libreoffice para Debian/Ubuntu, sudo dnf remove libreoffice para Fedora/Red Hat).

Vista de " Aplicaciones y características" en Windows 10/11 mostrando LibreOffice en la lista, listo para desinstalar.

Asegúrate de guardar todos tus documentos antes de desinstalar una suite ofimática. La desinstalación solo elimina el programa, no tus archivos personales.

¿Qué entra para examen?

A continuación, indicamos de este tema qué deberíais preparar para el examen:

  • Conocer los conceptos básicos de Informática y Ofimática.
  • Identificar los diferentes tipos de aplicaciones ofimáticas (procesadores de texto, hojas de cálculo, presentaciones, etc.) y ejemplos de cada uno.
  • Comprender la definición y los tipos de licencias de software (propietario, libre, comercial, shareware, freeware), así como los conceptos de Copyright, Dominio Público y Copyleft.
  • Conocer qué es una suite ofimática y diferenciar entre suites ofimáticas locales (ej. Microsoft Office, LibreOffice) y en la nube (ej. Google Workspace, Microsoft 365, OnlyOffice).
  • Identificar los datos que podrían usarse de forma segura en suites ofimáticas locales y cuáles, con las debidas precauciones, en suites ofimáticas en la nube, comprendiendo los riesgos de privacidad, seguridad y legales (como la ubicación de servidores fuera de la UE).
  • Entender las ventajas y desventajas de las suites en la nube, tomando como ejemplo Google Workspace (Google Docs, Sheets, Slides).
  • Conocer la importancia de la documentación y el soporte técnico, y dónde encontrarlos para aplicaciones generales y específicamente para Google Workspace.
  • Conocer las medidas básicas de ergonomía y descanso para trabajar con pantallas de visualización de datos (PVD), así como los principales riesgos laborales asociados (fatiga visual, trastornos musculoesqueléticos).
  • Saber qué es una “Cheat sheet” y la importancia de estas para ahorrar trabajo y mejorar la eficiencia.
  • Saber utilizar la búsqueda de Google con sus principales comandos (texto entre comillas, buscar extensiones de ficheros, incluir/excluir sitios donde buscar, establecer rangos de fechas, etc.) para poder afinar tus búsquedas.
  • Comprender las capacidades, ventajas y limitaciones de las búsquedas asistidas por Inteligencia Artificial (IA), y la importancia de contrastar la información generada por IA.
