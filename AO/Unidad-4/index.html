Unidad 4. Procesadores de Texto

Introducción

En esta unidad, exploraremos a fondo las funcionalidades de los procesadores de texto, herramientas indispensables para la creación, edición y formato de documentos escritos. Nos centraremos principalmente en Google Docs por su accesibilidad y capacidades de colaboración, pero también abordaremos conceptos aplicables a otras plataformas como Microsoft Word y Writer de OpenOffice.

1. Primer Vistazo a Google Docs

Google Docs es una herramienta de procesamiento de texto basada en la nube que permite crear y editar documentos en línea, colaborar en tiempo real con otros usuarios y acceder a tus archivos desde cualquier dispositivo con conexión a internet.

  • Acceso y Creación de Documentos: Para empezar, solo necesitas una cuenta de Google. Puedes acceder a Google Docs a través de tu navegador web o desde la aplicación móvil.
    • Crear un Nuevo Documento: Desde la página principal de Google Docs, haz clic en el botón "+" (Nuevo documento en blanco) o elige una plantilla preexistente.
    • Interfaz Principal: Familiarízate con la barra de menú superior (Archivo, Editar, Ver, Insertar, Formato, Herramientas, Extensiones, Ayuda) y la barra de herramientas de formato rápido debajo. El área central es donde se escribe y edita el texto.

2. Formato de Texto

El formato de texto es esencial para mejorar la legibilidad y la presentación visual de tus documentos.

2.1. Herramientas de formato

  • Fuente:
    • Tipo de Fuente: Elige entre una amplia variedad de fuentes (Arial, Times New Roman, Calibri, etc.) para establecer el estilo de tu texto.
    • Color: Modifica el color del texto para resaltar información o mejorar el contraste.
    • Fondo (Resaltado): Aplica un color de fondo al texto, similar a un rotulador, para destacar fragmentos importantes.
    • Tamaño de Fuente: Ajusta el tamaño del texto para títulos, subtítulos o cuerpo de texto, usando puntos (p. ej., 12pt, 14pt).
  • Estilos de Párrafo: Los estilos de párrafo aplican un conjunto predefinido de formatos (fuente, tamaño, color, espaciado) a bloques de texto. Ayudan a mantener la coherencia y a estructurar el documento.
    • Normal: Estilo predeterminado para el cuerpo del texto.
    • Título 1, Título 2, Subtítulo: Utilizados para jerarquizar los encabezados del documento.
    • Espaciado entre líneas: Ajusta la distancia vertical entre las líneas de texto (sencillo, 1.5, doble).
  • Otras Opciones de Formato:
    • Negrita (Bold), Cursiva (Italic), Subrayado (Underline): Para enfatizar palabras o frases.
    • Tachado: Para indicar texto que ha sido eliminado o no es válido.
    • Superíndice y Subíndice: Para fórmulas matemáticas (x²) o químicas (H₂O).
    • Borrar formato: Elimina todos los formatos aplicados a un texto seleccionado, dejándolo con el estilo normal.
  • Listas:
    • Listas Numeradas: Para elementos que siguen un orden específico (1., 2., 3.,...).
    • Listas con Viñetas: Para elementos no secuenciales (•, -, *...).
    • Listas Multinivel: Para organizar información de forma jerárquica con diferentes niveles de indentación y marcadores.
  • Tabulaciones: Permiten alinear texto a puntos específicos en una línea. Útiles para crear índices, directorios o formularios simples.
  • Sangrías:
    • Sangría de primera línea: La primera línea de un párrafo se mueve hacia la derecha.
    • Sangría francesa: Todas las líneas de un párrafo, excepto la primera, se mueven hacia la derecha.
    • Sangría izquierda y derecha: Ajusta el margen de todo el párrafo.

3. Formato Avanzado: Estilos y Temas

Los estilos y temas son herramientas poderosas para mantener la coherencia visual de tus documentos y aplicar cambios de formato de manera global.

3.1. Estilos y Conjuntos de Estilos

  • ¿Qué son los Estilos? Son colecciones predefinidas de características de formato (fuente, tamaño, color, espaciado, alineación) que se aplican a elementos específicos como títulos, subtítulos, cuerpo de texto, citas, etc.
  • Ventajas:
    • Consistencia: Aseguran que todos los elementos del mismo tipo (por ejemplo, todos los Títulos 1) tengan el mismo formato en todo el documento.
    • Edición Rápida: Si decides cambiar el estilo de todos los Títulos 2, solo modificas el estilo una vez y el cambio se aplica automáticamente a todas las instancias.
    • Navegación: En muchos procesadores de texto, los estilos de título se utilizan para generar automáticamente tablas de contenido y esquemas de navegación.
  • Aplicar y Modificar Estilos: En Google Docs, los encuentras en el menú "Formato" > "Estilos de párrafo". Puedes aplicar estilos predefinidos o crear y actualizar tus propios estilos.

3.2. Temas

  • ¿Qué son los Temas? Un tema es un conjunto coordinado de colores, fuentes y efectos visuales que se aplica a todo el documento para darle una apariencia profesional y unificada. Afecta no solo al texto, sino también a las tablas, gráficos y otros objetos.
  • Aplicar Temas: En Google Docs, puedes explorar y aplicar temas desde el menú "Formato" > "Temas".
  • Personalización: Puedes personalizar los colores y fuentes de un tema existente para adaptarlo a tus necesidades específicas o a la identidad corporativa.

4. Inserción de Objetos

Los procesadores de texto permiten enriquecer los documentos insertando diversos objetos.

4.1. Inserción de Imágenes

Las imágenes son fundamentales para ilustrar conceptos, embellecer el documento o presentar información visual.

Ejemplo: Opciones para insertar una imagen en Google Docs.

  • Herramienta de Imágenes:
    • Subir desde el ordenador: Inserta una imagen guardada en tu dispositivo.
    • Buscar en la web: Busca y descarga imágenes directamente desde el procesador de texto (Google Docs utiliza la búsqueda de Google).
    • Google Drive/Fotos: Inserta imágenes almacenadas en tus servicios de nube.
    • Por URL: Pega la dirección web de una imagen para insertarla.
    • Cámara: Toma una foto directamente con la cámara de tu dispositivo (si está disponible).
  • Opciones de Formato de Imágenes: Una vez insertada, puedes redimensionar, rotar, ajustar el brillo y contraste, recortar y aplicar estilos de ajuste de texto (en línea, ajustar texto, dividir texto, detrás del texto, delante del texto).
  • Dibujos: Permiten crear diagramas, flechas, formas o incluso dibujos a mano alzada directamente en el documento. Son útiles para visualizaciones rápidas o anotaciones.

4.2. Inserción de Símbolos y Texto

  • Símbolos Especiales: Para insertar caracteres que no se encuentran en el teclado estándar (p. ej., ©, ®, €, fracciones, símbolos matemáticos).
  • Cuadros de Texto: Permiten colocar texto en un cuadro flotante que puedes mover libremente por el documento, útil para citas, pies de foto o barras laterales.
  • WordArt (o Texto Estilizado): En algunos procesadores (como Word), permite crear texto con efectos visuales artísticos (sombras, reflejos, rotaciones). En Google Docs, esto se logra más a través de las opciones de formato avanzado del texto o con la herramienta de Dibujo.

5. Tablas

Las tablas son estructuras organizadas en filas y columnas que permiten presentar datos de forma clara y estructurada.

5.1 Creación de Tablas

  • Insertar Tabla: Puedes especificar el número de filas y columnas deseado al momento de la inserción.
  • Dibujar Tabla: Algunos procesadores (como Word) permiten dibujar la tabla manualmente, creando celdas de diferentes tamaños.

5.2 Herramientas de Tablas

Una vez creada la tabla, se activan herramientas específicas para su edición:

  • Insertar/Eliminar: Filas, columnas, celdas.
  • Combinar/Dividir Celdas: Unir varias celdas en una o dividir una celda en varias.
  • Bordes y Sombreado: Modificar el estilo, grosor y color de los bordes de la tabla, y aplicar sombreado a celdas o filas.
  • Alineación del Texto en Celdas: Alinear el contenido de las celdas horizontal (izquierda, centro, derecha) y verticalmente (arriba, medio, abajo).
  • Propiedades de la Tabla: Ajustar el ancho de las columnas, alto de las filas, espaciado entre celdas, etc.

6. Organización del Texto

Una buena organización es clave para la legibilidad y la profesionalidad de un documento extenso.

6.1 Portada

La portada es la primera página de un documento y suele contener información clave como el título, autor, fecha y quizás un logo o imagen.

  • Creación de Portadas: Muchos procesadores ofrecen plantillas de portada prediseñadas que puedes personalizar. También puedes crear una desde cero utilizando cuadros de texto, imágenes y formatos avanzados.

6.2. Configuración de la Página

  • Saltos:
    • Salto de Página: Fuerza el inicio de un nuevo contenido en la siguiente página.
    • Salto de Sección: Divide el documento en secciones, permitiendo aplicar diferentes formatos (márgenes, encabezados y pies de página, orientación) a cada una.
    • Salto de Columna: En documentos con varias columnas, fuerza el inicio del texto en la siguiente columna.
  • Página:
    • Orientación: Vertical (predeterminado) u Horizontal (apaisado).
    • Tamaño del Papel: A4, Carta, Legal, etc.
    • Color de la Página: Cambia el color de fondo de todo el documento.
  • Tabulaciones, Sangrías y Márgenes:
    • Márgenes: Establece el espacio en blanco alrededor de los bordes de la página (superior, inferior, izquierdo, derecho).
  • Herramientas de Encabezado y Pie de Página:
    • Encabezado: Texto o gráficos que aparecen en la parte superior de cada página (número de página, título del documento, nombre del autor).
    • Pie de Página: Texto o gráficos que aparecen en la parte inferior de cada página (número de página, fecha).
    • Opciones: Diferentes encabezados/pies de página para la primera página, páginas pares e impares, o por sección.

6.3. Tabla de Contenidos

Una tabla de contenidos (o índice) es una lista de los títulos y subtítulos del documento con sus respectivos números de página.

  • Generación Automática: Los procesadores de texto pueden generar automáticamente una tabla de contenidos basada en los estilos de título aplicados. Esto permite actualizarla fácilmente si se añade o elimina contenido.
  • Índices: Para documentos más extensos, se pueden crear índices de palabras clave o figuras.
  • Esquemas: La vista de esquema permite ver la estructura del documento en forma de jerarquía de títulos, facilitando la navegación y reorganización.
  • Notas al Pie: Texto explicativo o referencias que aparecen en la parte inferior de la página a la que se refieren.
  • Referencia Cruzada: Permite hacer referencia a otras partes del documento (p. ej., "Ver figura 3 en la página 15") y se actualiza automáticamente si el contenido se mueve.

7. Revisión y Autocorrección de Textos

Garantizar la precisión y la calidad del texto es fundamental.

7.1. Herramientas de Revisión

  • Ortografía y Gramática: Detecta errores ortográficos y gramaticales, ofreciendo sugerencias de corrección.
  • Contador de Palabras: Muestra el número de palabras, caracteres, párrafos y páginas del documento.
  • Diccionario de Sinónimos y Antónimos (Tesauro): Ayuda a encontrar palabras alternativas para enriquecer el vocabulario y evitar repeticiones.
  • Traducción: Algunos procesadores ofrecen la posibilidad de traducir el documento o partes de él.
  • Control de Cambios (o Sugerencias en Google Docs): Permite a múltiples usuarios realizar cambios y sugerencias en un documento, que luego pueden ser aceptados o rechazados por el autor.

7.2. Autocorrección

  • Reemplazo Automático: Corrige errores tipográficos comunes (p. ej., "teh" a "the") o reemplaza abreviaturas por texto completo (p. ej., "atm" a "en este momento").
  • Mayúsculas automáticas: Corrige automáticamente las mayúsculas iniciales de las frases.

8. Plantillas y Formularios

Las plantillas y formularios agilizan la creación de documentos estandarizados.

8.1. Creación de Documentos a Partir de una Plantilla

  • ¿Qué es una Plantilla? Es un modelo prediseñado de documento que ya incluye formato, estilos, diseños e incluso contenido de marcador de posición.
  • Ejemplos: Cartas, currículums, informes, folletos, facturas.
  • Ventajas: Ahorra tiempo, asegura la coherencia y profesionalidad.
  • Uso: Puedes acceder a galerías de plantillas dentro del procesador de texto o descargar plantillas de terceros.

8.2. Creación de Nuevas Plantillas

Puedes guardar un documento con el formato y contenido que desees como una plantilla para reutilizarlo en el futuro. Esto es útil para documentos recurrentes con un diseño específico.

8.3. Formularios

  • Elementos de Formulario: Permiten crear campos interactivos en un documento donde los usuarios pueden introducir información (casillas de verificación, cuadros de texto, menús desplegables, botones de opción).
  • Usos: Encuestas, solicitudes de información, cuestionarios.
  • Protección: Puedes proteger el formulario para que solo se puedan editar los campos específicos, evitando modificaciones accidentales al resto del documento.

9. Gestión de Correspondencia

La gestión de correspondencia se refiere a las herramientas para crear y enviar documentos personalizados a múltiples destinatarios.

9.1. Diseño de Sobres y Etiquetas

  • Combinación de Correspondencia (Mail Merge): Es una función potente que permite combinar una fuente de datos (como una lista de nombres y direcciones en una hoja de cálculo) con un documento principal (una carta, sobre o etiqueta). El procesador de texto genera automáticamente una copia personalizada del documento para cada entrada en la fuente de datos.
  • Creación de Sobres: Permite ingresar la dirección del destinatario y del remitente, y el procesador de texto lo formatea para imprimirlo directamente en un sobre.
  • Creación de Etiquetas: Útil para generar etiquetas de envío, de dirección o de identificación de productos en grandes cantidades.

10. Creación y Uso de Macros (Word)

  • ¿Qué es una Macro? Una macro es una secuencia de comandos o acciones grabadas que se pueden ejecutar con un solo clic o atajo de teclado. Automatizan tareas repetitivas.
  • Ejemplos: Formatear texto de una manera específica, insertar bloques de texto complejos, generar informes.
  • Grabación de Macros: Puedes grabar tus acciones para que el procesador de texto (principalmente Word) las convierta en código VBA (Visual Basic for Applications).
  • Ejecución de Macros: Asigna un botón o un atajo de teclado a la macro para ejecutarla rápidamente.
  • Advertencia: Las macros pueden contener virus, por lo que es importante ser cauteloso al abrir documentos con macros de fuentes desconocidas.

11. Alternativas: Writer OpenOffice

Aunque nos hemos centrado en Google Docs, es importante conocer otras opciones.

  • Writer (Apache OpenOffice/LibreOffice): Es un procesador de texto de código abierto y gratuito que ofrece funcionalidades similares a Microsoft Word. Es una excelente alternativa para quienes buscan una suite de oficina completa sin costo.
  • Ventajas: Gratuito, compatible con formatos de archivo de Word (.docx), buena suite de funcionalidades.
  • Interfaz: Similar a versiones antiguas de Word, lo que facilita la transición para muchos usuarios.

12. Importación, Exportación e Impresión de Datos

La interoperabilidad es clave en el manejo de documentos.

12.1. Conversión entre Distintos Formatos

  • Exportar: Guardar un documento en un formato diferente al original (p. ej., de .docx a .pdf, de Google Docs a .odt o .rtf).
  • Importar: Abrir un documento creado en un formato diferente (p. ej., abrir un archivo .docx en Google Docs o Writer).
  • Formatos Comunes:
    • .docx, .doc: Formatos de Microsoft Word.
    • .odt: Formato de OpenDocument Text (usado por Writer).
    • .pdf: Formato de Documento Portátil (Portable Document Format), ideal para compartir documentos que deben conservar su formato original y no ser editados fácilmente.
    • .rtf: Formato de Texto Enriquecido (Rich Text Format), compatible con la mayoría de los procesadores de texto.
    • .txt: Texto plano, sin formato.

12.2. Importación de Texto a Través de Otros Dispositivos

  • Copiar y Pegar: La forma más sencilla de transferir texto.
  • Sincronización en la Nube: Google Docs (con Google Drive), OneDrive (con Word) permiten acceder y editar documentos desde cualquier dispositivo con conexión a internet.
  • Dispositivos de Almacenamiento: USB, discos externos.

12.3. Impresión

  • Configuración de Impresión: Ajustar el número de copias, rango de páginas, orientación, tamaño de papel, impresión a doble cara, márgenes.
  • Vista Previa de Impresión: Visualiza cómo se verá el documento impreso antes de enviarlo a la impresora para evitar errores.
  • Opciones Avanzadas: Impresión de folletos, impresión a escalas de grises, selección de impresora.

13. Aspectos de Seguridad y Trabajo en Equipo

Los procesadores de texto modernos están diseñados para la colaboración y la seguridad.

13.1. Herramientas de Trabajo en Equipo

  • Compartir y Colaborar:
    • Permisos: Define quién puede ver, comentar o editar el documento.
    • Enlaces para Compartir: Genera enlaces públicos o restringidos.
  • Comentarios: Permiten a los colaboradores añadir notas o preguntas sobre partes específicas del documento sin modificar el texto principal.
  • Modo Sugerencias (Google Docs) / Control de Cambios (Word): Registra todas las modificaciones realizadas por los colaboradores, que el autor puede revisar y aceptar o rechazar individualmente.
  • Historial de Versiones: Permite ver versiones anteriores del documento, quién hizo los cambios y cuándo. Es invaluable para recuperar contenido o revertir a una versión anterior.
  • Chat en Documento: En Google Docs, puedes chatear con otros colaboradores en tiempo real dentro del mismo documento.

13.2. Herramientas de Seguridad

  • Contraseñas: Protege documentos con contraseña para restringir el acceso a usuarios no autorizados (más común en Word).
  • Cifrado: Algunos procesadores ofrecen opciones de cifrado para proteger el contenido del documento.
  • Restricción de Edición: Limita lo que los usuarios pueden editar en un documento (p. ej., solo rellenar formularios, solo hacer comentarios).
  • Firma Digital: Permite autenticar la autoría de un documento y verificar que no ha sido alterado después de la firma.
  • Copias de Seguridad: La mayoría de los servicios en la nube (Google Drive, OneDrive) realizan copias de seguridad automáticas de tus documentos, protegiéndolos contra pérdidas.
