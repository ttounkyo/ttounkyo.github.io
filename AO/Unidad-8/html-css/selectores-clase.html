Selectores de clase

Un selector de clase permite aplicar estilos a uno o varios elementos que comparten una misma clase. Se indica con un punto (.) seguido del nombre de la clase.

Ventajas

  • Permiten reutilizar estilos en diferentes elementos.
  • Dan mayor flexibilidad que los selectores de etiqueta.

Ejemplo de uso

.importante {
    background-color: yellow;
    font-weight: bold;
}
