Actividad 3

Crea un documento web (“index.html”) que debe cargar una imagen llamada “imagen.jpg”, la cual está ubicada en un directorio llamado “img”. Escriba el código HTML del documento teniendo en cuenta las siguientes especificaciones:

  • La imagen tendrá las siguientes dimensiones: Anchura: 558 px (width) y Altura: 600 px (height).
  • Cuando se posicione el cursor sobre la imagen, debe aparecer el texto: “Máquina para mecanizar piezas”.
  • Si la imagen no se carga correctamente, aparecerá el texto alternativo: “Imagen de una máquina para mecanizar piezas”.
  • El documento debe ser compatible con los caracteres propios de Europa Occidental.
  • El título del documento será: “Ejemplo de documento con imagen”.
