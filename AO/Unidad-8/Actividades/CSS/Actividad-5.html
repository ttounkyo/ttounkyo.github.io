Actividad 5 - Diseño de estructura con floats

Objetivo: Practicar el uso de contenedores <div> combinados con la propiedad CSS float para organizar el diseño de una página.

Instrucciones:

  • Crea una estructura similar a las propuestas de ejemplo usando <div> para cada bloque de contenido.
  • Asigna diferentes clases para controlar el ancho, alto y color de fondo de cada sección.
  • Utiliza la propiedad float en CSS para posicionar los elementos horizontalmente.
  • Aplica una segunda fila de elementos respetando el uso de floats y el orden indicado.
  • Puedes elegir entre:
    • Usar clases personalizadas como .uno, .dos, .tres, etc.
    • O emplear clases genéricas como .w-20, .red, .blue, etc.

Recomendaciones:

  • Utiliza porcentajes para definir los anchos (20%, 30%, 40%, ...).
  • Añade títulos dentro de cada <div> usando etiquetas <h2> para identificar cada bloque.
  • Asegúrate de limpiar los floats al final de cada fila si es necesario (puedes usar la propiedad clear o un clearfix).

Pistas:

  • Para dividir una fila en dos mitades, usa dos <div> con height: 3em dentro de un contenedor con height: 6em.
  • Los colores de fondo ayudan a visualizar mejor la estructura.
