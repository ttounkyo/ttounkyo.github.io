Enlaces e Imágenes HTML
Elemento Descripción
<a> Define un hipervínculo.
<a href="https://ejemplo.com" target="_blank">Enlace</a>
Atributos comunes: href, target, rel, download
<img> Inserta una imagen.
<img src="imagen.jpg" alt="Descripción" width="300" height="200">
Atributos comunes: src, alt, width, height, loading
<figure> Contenedor para imágenes con leyenda.
<figure> <img src="imagen.jpg" alt="Descripción"> <figcaption>Leyenda de la imagen</figcaption> </figure>
<picture> Contenedor para múltiples recursos de imagen.
<picture> <source media="(min-width: 800px)" srcset="imagen-grande.jpg"> <img src="imagen-pequeña.jpg" alt="Descripción"> </picture>

Formularios HTML

<label for="nombre">Nombre:</label> <input id="nombre" type="text">
<fieldset> Agrupa elementos relacionados en un formulario.
<fieldset> <legend>Datos personales</legend> ... </fieldset>

CSS3: Flexbox
Propiedad Descripción y Valores
display Activa el modelo flexbox.
display: flex; /* o inline-flex */
flex-direction Dirección del eje principal.
flex-direction: row | row-reverse | column | column-reverse;
flex-wrap Control del desbordamiento de los elementos.
flex-wrap: nowrap | wrap | wrap-reverse;
flex-flow Abreviatura para flex-direction y flex-wrap.
flex-flow: row wrap;
justify-content Alineación en el eje principal.
justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly;
align-items Alineación en el eje transversal.
align-items: stretch | flex-start | flex-end | center | baseline;
align-content Distribución de líneas en el eje transversal.
align-content: flex-start | flex-end | center | space-between | space-around | stretch;
flex Abreviatura para flex-grow, flex-shrink y flex-basis.
flex: 1 0 auto; /* grow shrink basis */
order Controla el orden de los elementos.
order: 2; /* por defecto es 0 */
CSS3: Grid Layout
Propiedad Descripción y Valores
display Activa el modelo grid.
display: grid | inline-grid;
grid-template-columns Define las columnas de la cuadrícula.
grid-template-columns: 100px 1fr 2fr;
grid-template-columns: repeat(3, 1fr);
grid-template-rows Define las filas de la cuadrícula.
grid-template-rows: auto 200px 1fr;
grid-template-areas Define áreas con nombre.
grid-template-areas: "header header" "sidebar content" "footer footer";
grid-column-gap Espacio entre columnas (obsoleto, usar column-gap).
column-gap: 20px;
grid-row-gap Espacio entre filas (obsoleto, usar row-gap).
row-gap: 20px;
gap Abreviatura para row-gap y column-gap.
gap: 10px 20px; /* fila columna */
grid-column Especifica posición en columnas.
grid-column: 1 / 3; /* inicio / final */
grid-column: 1 / span 2; /* inicio / extensión */
grid-row Especifica posición en filas.
grid-row: 2 / 4;
grid-area Nombre del área o abreviatura para posicionamiento.
grid-area: header;
grid-area: 1 / 1 / 2 / 3; /* fila-inicio / col-inicio / fila-fin / col-fin */
CSS3: Transformaciones y Animaciones
Propiedad Descripción y Valores
transform Aplica transformaciones 2D o 3D.
transform: translate(50px, 20px);
transform: rotate(45deg);
transform: scale(1.5);
transform: skew(10deg, 5deg);
transition Define transiciones entre estados.
transition: property duration timing-function delay;
transition: all 0.3s ease-in-out;
animation Configura animaciones.
animation: name duration timing-function delay iteration-count direction fill-mode;
animation: slideIn 2s ease-in-out infinite;
@keyframes Define los estados de una animación.
@keyframes slideIn { 0% { transform: translateX(-100%); } 100% { transform: translateX(0); } }
CSS3: Selectores Avanzados
Selector Descripción y Ejemplo
Elemento Selecciona todos los elementos del tipo especificado.
p { color: blue; }
Clase Selecciona elementos con clase específica.
.nota { font-style: italic; }
ID Selecciona un elemento con ID específico.
#cabecera { background-color: black; }
Atributo Selecciona elementos con atributo específico.
input[type="text"] { border: 1px solid gray; }
a[href^="https"] { color: green; } /* Comienza con */
a[href$=".pdf"] { font-weight: bold; } /* Termina con */
a[href*="ejemplo"] { text-decoration: underline; } /* Contiene */
Descendiente Selecciona elementos descendientes.
article p { line-height: 1.6; }
Hijo directo Selecciona elementos hijo directo.
ul > li { list-style-type: square; }
Adyacente Selecciona elemento adyacente.
h2 + p { margin-top: 0; }
Hermano general Selecciona elementos hermanos.
h2 ~ p { color: gray; }
Pseudo-clases Selecciona elementos en estados específicos.
a:hover { text-decoration: underline; }
input:focus { border-color: blue; }
li:first-child { font-weight: bold; }
p:nth-child(odd) { background: #f5f5f5; }
Pseudo-elementos Selecciona partes específicas de un elemento.
p::first-line { font-variant: small-caps; }
p::first-letter { font-size: 2em; }
p::before { content: "→ "; }
p::after { content: " ←"; }
CSS3: Media Queries y Responsive Design
Propiedad/Regla Descripción y Valores
@media Define reglas según características del dispositivo.
@media screen and (max-width: 768px) { /* Estilos para dispositivos con ancho máximo de 768px */ body { font-size: 14px; } }
@media print { /* Estilos para impresión */ .no-print { display: none; } }
viewport Meta etiqueta para control de visualización en dispositivos móviles.
<meta name="viewport" content="width=device-width, initial-scale=1.0">
min-width, max-width Media features para consultas basadas en ancho.
@media (min-width: 576px) and (max-width: 992px) { ... }
orientation Media feature para orientación del dispositivo.
@media (orientation: landscape) { ... }
CSS3: Variables y Unidades
Propiedad/Característica Descripción y Valores
Variables CSS (Custom Properties) Almacena valores reutilizables.
:root { --color-primario: #e67e22; --fuente-principal: 'Arial', sans-serif; }
h1 { color: var(--color-primario); font-family: var(--fuente-principal); }
Unidades relativas Unidades que dependen de otros valores.
em: relativa al tamaño de fuente del elemento padre
rem: relativa al tamaño de fuente del elemento raíz
%: porcentaje relativo al elemento padre
vw: 1% del ancho de la ventana
vh: 1% del alto de la ventana
vmin: 1% de la dimensión más pequeña
vmax: 1% de la dimensión más grande
Unidades absolutas Unidades con tamaño fijo.
px: píxeles
pt: puntos (1/72 de pulgada)
in: pulgadas
cm: centímetros
mm: milímetros
Funciones CSS Funciones para valores dinámicos.
calc(): cálculos matemáticos
calc(100% - 40px)
rgb(), rgba(): colores RGB/RGBA
hsl(), hsla(): colores HSL/HSLA
url(): ruta a recurso
