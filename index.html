Resumen de HTML Básico

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado que se utiliza para crear y estructurar páginas web. Organiza el contenido como títulos, párrafos, imágenes, enlaces y más.

Ejemplo básico de HTML

<!DOCTYPE html>
<html>
<head>
  <title>Mi primera página</title>
</head>
<body>
  <h1>¡Hola, mundo!</h1>
  <p>Esta es mi primera página web en HTML.</p>
</body>
</html>

Explicación del ejemplo:

  • <!DOCTYPE html> define que el documento es HTML5
  • <html> es el elemento raíz de la página
  • <head> contiene información sobre el documento
  • <title> establece el título del documento
  • <body> contiene los elementos visibles
  • <h1> define el titular de la página
  • <p> define un párrafo

HTML Tags (Etiquetas HTML)

Las etiquetas HTML se escriben entre corchetes angulares y definen los distintos elementos de una página.

<etiqueta>Contenido</etiqueta>
  • Las etiquetas vienen en pares: <p> y </p>
  • La etiqueta de cierre incluye una barra: </nombre>

Navegadores

Los navegadores como Chrome, Firefox, Edge o Safari leen documentos HTML y los muestran en pantalla.

  • No muestran las etiquetas, pero las usan para estructurar el contenido.

La declaración <!DOCTYPE>

Esta declaración indica al navegador que se está utilizando HTML5 y debe colocarse al principio del archivo HTML.

  • Aparece una sola vez, antes de la etiqueta <html>
  • No distingue entre mayúsculas y minúsculas
  • Versión HTML5: <!DOCTYPE html>
