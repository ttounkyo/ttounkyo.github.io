Introducción a CSS

CSS (Cascading Style Sheets o Hojas de Estilo en Cascada) es un lenguaje utilizado para describir la presentación y estilo de un documento HTML. Permite modificar colores, fuentes, márgenes, posiciones, tamaños y mucho más.

Gracias a CSS podemos separar el contenido (HTML) de la presentación (diseño), logrando sitios web más limpios, reutilizables y fáciles de mantener.

Formas de incluir CSS en un documento HTML

Método Descripción Ejemplo
CSS en línea Se aplica directamente dentro del atributo style de un elemento HTML.
<p style="color: red;">Texto rojo</p>
CSS interno Se escribe dentro de una etiqueta <style> en el <head>.
<style>
  p { color: green; }
</style>
CSS externo Se enlaza un archivo .css desde el <head> del documento.
<link rel="stylesheet" href="estilos.css">

Ejemplo con CSS interno

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Ejemplo CSS</title>
    <style>
        body {
            background-color: #f0f0f0;
            font-family: Arial, sans-serif;
        }
        h1 {
            color: navy;
        }
    </style>
</head>
<body>
    <h1>Hola, mundo CSS</h1>
    <p>Este es un párrafo con estilo.</p>
</body>
</html>

Este ejemplo demuestra cómo aplicar estilos directamente dentro del archivo HTML usando la etiqueta <style>.
