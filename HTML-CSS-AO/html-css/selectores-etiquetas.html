Selectores de etiqueta (o de tipo)

Los selectores de etiqueta aplican estilos a todos los elementos HTML de un tipo específico, como p, h1 o a.

Ventajas

  • Fáciles de usar.
  • Permiten aplicar estilos generales rápidamente.
  • Útiles para mantener un estilo uniforme.

Limitaciones

  • Demasiado generales en algunos casos.
  • No sirven para personalizaciones específicas.

Ejemplo de uso

h1 {
    color: navy;
    font-family: Arial, sans-serif;
}

p {
    font-size: 16px;
    line-height: 1.5;
}

a {
    color: green;
    text-decoration: none;
}
<h1>Título principal</h1>
<p>Este es un párrafo de texto normal.</p>
<a href="#">Enlace sin subrayado</a>

Selector combinado

Puedes combinar una etiqueta con una clase para mayor precisión:

p.intro {
    font-weight: bold;
}
<p class="intro">Este párrafo es introductorio.</p>
