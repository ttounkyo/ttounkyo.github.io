Selectores de ID

El selector de ID se usa para aplicar estilos a un único elemento identificado por su atributo id. Se indica con una almohadilla (#) seguida del nombre del ID.

Importante

El ID debe ser único en cada documento HTML.

Ejemplo de uso

#cabecera {
    background-color: lightgray;
    padding: 10px;
}
<header id="cabecera">Encabezado principal</header>
