<a> Define un hipervínculo básico.
Atributos comunes: href, target, rel, download
<a href="https://ejemplo.com">Enlace a ejemplo.com</a>
<a target="_blank"> Abre el enlace en una nueva pestaña o ventana.
Considera añadir rel="noopener noreferrer" por seguridad
 <a href="https://ejemplo.com" target="_blank">Abrir en nueva pestaña</a>
<a download> Crea un enlace para descargar un archivo.
El atributo download puede especificar un nombre de archivo personalizado

El destino debe tener un id coincidente

Parámetros: subject, cc, bcc, body

<a href="mailto:correo@ejemplo.com?subject=Asunto&body=Contenido">Email con asunto</a>
<a tel:> Crea un enlace para llamar a un número de teléfono.
Funciona principalmente en móviles
 <a href="tel:+34123456789">Llamar ahora</a>
<a> con imagen Utiliza una imagen como enlace clickeable.
Cualquier contenido puede estar dentro de un enlace
 <a href="https://ejemplo.com">
  <img src="logo.png" alt="Logo de Ejemplo">
</a>
<nav> con <a> Grupo de enlaces semánticos para navegación.
Ayuda a los lectores de pantalla
<nav>

<button> vs <a> Diferencia entre botones y enlaces.
Usa <button> para acciones, <a> para navegación
 <button type="button">Acción en la página</button>

href Especifica la URL de destino del enlace.
Puede ser absoluta, relativa o anclaje
 <a href="https://ejemplo.com">Enlace</a>
<a href="pagina.html">Relativo</a>
<a href="#seccion">Anclaje</a>
target Define dónde se abre el enlace.
Valores: _blank, _self, _parent, _top

<a href="url" target="_self">Misma ventana</a>

Valores: nofollow, noopener, noreferrer, etc.

Etiqueta <a>

La etiqueta <a> (anchor) se utiliza para crear enlaces en una página web. Es una etiqueta que requiere una etiqueta de cierre y debe incluir al menos el atributo href.

Ejemplo:

<a href="https://ejemplo.com">Texto del enlace</a>

Atributos principales de <a>:

  • href: Especifica la URL de destino. Puede ser relativa (ej: contacto.html), absoluta (ej: https://ejemplo.com), anclaje interno (ej: #seccion1), o protocolos especiales (ej: mailto:, tel:).
  • target: Define dónde se abrirá el enlace (_blank, _self, _parent, _top).
  • rel: Establece la relación entre la página actual y la enlazada (ej: nofollow, noopener, noreferrer).
  • download: Indica que el enlace descargará un archivo.
  • title: Texto que aparece al pasar el cursor por encima.

Ejemplo completo:

<a href="https://ejemplo.com" target="_blank" rel="noopener noreferrer" title="Visita nuestra página">Visitar Ejemplo.com</a>

Tipos de enlaces:

1. Enlaces a páginas web:

<a href="https://ejemplo.com">Visitar sitio externo</a><a href="contacto.html">Ir a página de contacto</a>

2. Enlaces internos con anclajes:

<a href="#seccion1">Ir a la sección 1</a>
<!-- En otra parte del documento: -->
<h2 id="seccion1">Sección 1</h2>

3. Enlaces para correo electrónico:

<a href="mailto:info@ejemplo.com">Enviar correo</a>
<a href="mailto:info@ejemplo.com?subject=Consulta&body=Hola,%20quiero%20información">Enviar consulta</a>

4. Enlaces para teléfono:

<a href="tel:+34612345678">Llamar ahora: +34 612 345 678</a>

5. Enlaces de descarga:

<a href="documentos/manual.pdf" download>Descargar manual</a>
    <a href="documentos/manual.pdf" download="manual-usuario.pdf">Descargar con nombre personalizado</a>

Enlaces con imágenes:

<a href="https://ejemplo.com">
    <img src="logo.png" alt="Logo de Ejemplo">
</a>

Enlaces dentro de listas:

<ul>
    <li><a href="index.html">Inicio</a></li>
    <li><a href="servicios.html">Servicios</a></li>
    <li><a href="contacto.html">Contacto</a></li>
</ul>

Enlaces con atributos de seguridad:

<a href="https://sitio-externo.com" target="_blank" rel="noopener noreferrer">Sitio seguro</a>

Importante:

  • Usa textos descriptivos en los enlaces, evita "haz clic aquí" o "más información".
  • Los enlaces deben ser visualmente distinguibles del texto normal (normalmente subrayados y en color azul por defecto).
  • Añade rel="noopener noreferrer" cuando uses target="_blank" por seguridad.
  • Los enlaces internos con # son útiles para crear menús de navegación en páginas largas.

Ejemplo del uso de enlaces en HTML

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Enlaces en HTML</title>
</head>
<body>

    <nav>
        <a href="#inicio">Inicio</a>
        <a href="#servicios">Servicios</a>
        <a href="#contacto">Contacto</a>
    </nav>

    <section id="inicio">
        <h2>Inicio</h2>
        <p>Bienvenido a nuestra página. <a href="about.html">Conoce más sobre nosotros</a>.</p>
    </section>

    <section id="servicios">
        <h2>Servicios</h2>
        <ul>
            <li><a href="diseno.html">Diseño web</a></li>
            <li><a href="desarrollo.html">Desarrollo</a></li>
            <li><a href="seo.html">SEO</a></li>
        </ul>
    </section>

    <section id="contacto">
        <h2>Contacto</h2>
        <p>Puedes contactarnos por:</p>
        <p>
            <a href="mailto:info@ejemplo.com">info@ejemplo.com</a><br>
            <a href="tel:+34612345678">+34 612 345 678</a>
        </p>
        <p>
            <a href="contacto.html" class="button-link">Formulario de contacto</a>
        </p>
    </section>

    <p>Descarga nuestro <a href="recursos/catalogo.pdf" download>catálogo completo</a>.</p>

    <p>
        <a href="https://ejemplo.com" target="_blank" rel="noopener noreferrer">
            <img src="logo.png" alt="Logo de Ejemplo" width="150">
        </a>
    </p>

</body>
</html>
