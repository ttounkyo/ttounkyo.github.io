Elementos Estructurales en HTML

Elemento Descripción Ejemplo
<html> Elemento raíz de un documento HTML. <html lang="es">...</html>
<head> Contiene metadatos y enlaces a recursos externos. <head>...</head>
<title> Define el título del documento. <title>Mi página web</title>
<body> Contiene el contenido visible del documento. <body>...</body>
<header> Define la cabecera de una sección o página. <header>...</header>
<nav> Define un bloque de navegación. <nav>...</nav>
<main> Contenido principal del documento. <main>...</main>
<section> Define una sección del documento. <section>...</section>
<article> Define contenido independiente y autocontenido. <article>...</article>
<aside> Define contenido secundario relacionado. <aside>...</aside>
<footer> Define el pie de una sección o página. <footer>...</footer>
<div> Contenedor genérico para agrupar contenido. <div class="container">...</div>

Ejemplo del uso de etiquetas de estructura

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi página web</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <header>
        <h1>Bienvenidos a mi sitio</h1>
        <nav>
            <ul>
                <li><a href="#">Inicio</a></li>
                <li><a href="#">Acerca de</a></li>
                <li><a href="#">Contacto</a></li>
            </ul>
        </nav>
    </header>

    <main>
        <section>
            <h2>Sección principal</h2>
            <p>Este es el contenido principal de la página.</p>
        </section>

        <aside>
            <h3>Contenido relacionado</h3>
            <p>Información complementaria aquí.</p>
        </aside>
    </main>

    <footer>
        <p>&copy; 2025 Mi Sitio Web</p>
    </footer>
</body>
</html>
