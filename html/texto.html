Elementos de Texto HTML

Elemento Descripción
<h1> a <h6> Encabezados de diferentes niveles.
<h1>Título principal</h1>
<p> Define un párrafo.
<p>Este es un párrafo.</p>
<span> Contenedor en línea para texto.
<span class="highlight">texto destacado</span>
<strong> Texto importante con énfasis fuerte.
<strong>Texto importante</strong>
<em> Texto enfatizado.
<em>Texto enfatizado</em>
<mark> Texto resaltado o marcado.
<mark>Texto marcado</mark>
<code> Fragmento de código.
<code>var x = 5;</code>
<pre> Texto preformateado (mantiene espacios y saltos).
<pre> Texto con formato </pre>
<blockquote> Cita en bloque.
<blockquote>Cita extensa...</blockquote>
<q> Cita corta en línea.
<q>Cita corta</q>

Ejemplo del uso de etiquetas de texto

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Etiquetas de texto en HTML</title>
</head>
<body>
    <h1>Encabezado de nivel 1</h1>
    <h2>Encabezado de nivel 2</h2>
    <h3>Encabezado de nivel 3</h3>
    <h4>Encabezado de nivel 4</h4>
    <h5>Encabezado de nivel 5</h5>
    <h6>Encabezado de nivel 6</h6>

    <p>Este es un párrafo normal. Puede contener texto <strong>en negrita</strong> y texto <em>en cursiva</em>.</p>

    <p>También puedes usar <b>negrita</b> con la etiqueta b y <i>cursiva</i> con la etiqueta i.</p>

    <p>Texto <mark>resaltado</mark> con la etiqueta mark.</p>

    <p>Un texto <small>más pequeño</small> con la etiqueta small.</p>

    <p>Texto <del>tachado</del> con la etiqueta del.</p>

    <p>Texto <ins>subrayado</ins> con la etiqueta ins.</p>

    <p>Texto <sub>subíndice</sub> y texto <sup>superíndice</sup>.</p>

    <blockquote>
        Esta es una cita de bloque con la etiqueta blockquote. Se utiliza para citar contenido de otras fuentes.
    </blockquote>

    <p>También puedes usar <q>citas cortas en línea</q> con la etiqueta q.</p>

    <pre>
        Este es texto preformateado.
        Conserva    los espacios    y
        saltos de línea.
    </pre>

    <code>Este es un fragmento de código</code>

    <p>Definición de <abbr title="HyperText Markup Language">HTML</abbr> usando la etiqueta abbr.</p>

    <address>
        Escrito por: Juan Pérez<br>
        Contacto: <a href="mailto:juan@ejemplo.com">juan@ejemplo.com</a>
    </address>
</body>
</html>
