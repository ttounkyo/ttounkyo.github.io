Elementos de Imágenes en HTML

Elemento Descripción
<a> Define un hipervínculo.
<a href="https://ejemplo.com" target="_blank">Enlace</a>
Atributos comunes: href, target, rel, download
<img> Inserta una imagen.
<img src="imagen.jpg" alt="Descripción" width="300" height="200">
Atributos comunes: src, alt, width, height, loading
<figure> Contenedor para imágenes con leyenda.
<figure> <img src="imagen.jpg" alt="Descripción"> <figcaption>Leyenda de la imagen</figcaption> </figure>
<picture> Contenedor para múltiples recursos de imagen.
<picture> <source media="(min-width: 800px)" srcset="imagen-grande.jpg"> <img src="imagen-pequeña.jpg" alt="Descripción"> </picture>
