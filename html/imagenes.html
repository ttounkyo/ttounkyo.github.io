Elementos de Imágenes en HTML

Elemento Descripción Ejemplo Atributos
<a> Define un hipervínculo. <a href="https://ejemplo.com" target="_blank">Enlace</a> href, target, rel, download
<img> Inserta una imagen. <img src="imagen.jpg" alt="Descripción" width="300" height="200"> src, alt, width, height, loading
<figure> Contenedor para imágenes con leyenda. <figure> <img src="imagen.jpg" alt="Descripción"> <figcaption>Leyenda de la imagen</figcaption> </figure> —
<picture> Contenedor para múltiples recursos de imagen. <picture> <source media="(min-width: 800px)" srcset="imagen-grande.jpg"> <img src="imagen-pequeña.jpg" alt="Descripción"> </picture> media, srcset

Etiqueta <img>

La etiqueta <img> se utiliza para insertar imágenes en una página web. Es una etiqueta vacía (no tiene etiqueta de cierre) y debe incluir al menos el atributo src.

Ejemplo:

<img src="ruta/imagen.jpg" alt="Descripción de la imagen">

Atributos principales de <img>:

  • src: Especifica la ruta de la imagen. Puede ser relativa (ej: imágenes/logo.png) o absoluta (ej: https://ejemplo.com/logo.png).
  • alt: Texto alternativo que se muestra si la imagen no se puede cargar. Es importante para la accesibilidad.
  • width: Define el ancho de la imagen (en píxeles o %).
  • height: Define la altura de la imagen.
  • title: Texto que aparece al pasar el cursor por encima.

Ejemplo completo:

<img src="imagenes/perro.jpg" alt="Un perro jugando" width="300" height="200" title="Perro feliz">

Formato dentro de listas:

Puedes usar imágenes dentro de elementos de lista:

<ul>
    <li>
        <img src="iconos/sol.png" alt="Sol"> Día soleado
    </li>
    <li>
        <img src="iconos/lluvia.png" alt="Lluvia"> Día lluvioso
    </li>
</ul>

Imágenes como enlaces:

<a href="https://example.com"> <img src="boton.png" alt="Ir a ejemplo"> </a>

Importante:

  • Siempre incluir el atributo alt para accesibilidad.
  • No uses imágenes decorativas sin alt o usa alt="" si no aportan información.
  • Asegúrate de que el archivo existe en la ruta indicada.

Ejemplo del uso de imágenes en HTML

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Imágenes en HTML</title>
</head>
<body>

    <h2>Etiqueta <a> para enlaces</h2>
    <a href="https://ejemplo.com" target="_blank">Visitar Ejemplo.com</a>

    <h2>Etiqueta <img> para insertar imágenes</h2>
    <img src="imagen.jpg" alt="Descripción de la imagen" width="300" height="200">

    <h2>Etiqueta <figure> con leyenda</h2>
    <figure>
        <img src="imagen.jpg" alt="Descripción">
        <figcaption>Leyenda de la imagen</figcaption>
    </figure>

    <h2>Etiqueta <picture> para imágenes responsivas</h2>
    <picture>
        <source media="(min-width: 800px)" srcset="imagen-grande.jpg">
        <img src="imagen-pequeña.jpg" alt="Imagen alternativa">
    </picture>

</body>
</html>

Este bloque muestra cómo trabajar con imágenes, enlaces y estructuras responsivas usando <a>, <img>, <figure> y <picture>.
