Elementos de Listas

Listas no ordenadas (<ul>)

La etiqueta <ul> (Unordered List) se utiliza para crear listas sin un orden específico, donde cada elemento está marcado con un punto o viñeta.

<ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
    <li>Elemento 3</li>
</ul>

Listas ordenadas (<ol>)

La etiqueta <ol> (Ordered List) crea listas numeradas o con otro tipo de ordenación. Puedes personalizar el tipo y el número de inicio:

<ol type="1" start="3">
    <li>Elemento 3</li>
    <li>Elemento 4</li>
</ol>

Tipos de numeración en listas ordenadas:

  • type="1": Números (1, 2, 3...)
  • type="A": Letras mayúsculas (A, B, C...)
  • type="a": Letras minúsculas (a, b, c...)
  • type="I": Números romanos mayúsculos (I, II, III...)
  • type="i": Números romanos minúsculos (i, ii, iii...)

Elementos de lista (<li>)

La etiqueta <li> (List Item) se usa para definir cada elemento dentro de una lista. Siempre debe estar dentro de un elemento padre <ul>, <ol> o <menu>. 

<li>Elemento de lista</li>

Listas de definiciones (<dl>)

La etiqueta <dl> (Definition List) se utiliza para crear listas de términos y sus definiciones:

<dl>
    <dt>Término</dt>
    <dd>Definición</dd>
    <dt>Otro término</dt>
    <dd>Otra definición</dd>
</dl>

Importante al formatear listas:

  • Siempre cierra cada etiqueta <li> con </li>
  • Asegúrate de que los elementos <li> estén siempre dentro de un elemento contenedor (<ul>, <ol> o <menu>)
  • Para las listas de definición, usa siempre <dt> para el término y <dd> para la definición

Ejemplo del uso de listas en HTML

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Listas en HTML</title>
</head>
<body>

    <h2>Lista no ordenada</h2>
    <ul>
        <li>Manzana</li>
        <li>Banana</li>
        <li>Cereza</li>
    </ul>

    <h2>Lista ordenada</h2>
    <ol type="A" start="2">
        <li>Segundo elemento</li>
        <li>Tercer elemento</li>
    </ol>

    <h2>Lista de definiciones</h2>
    <dl>
        <dt>HTML</dt>
        <dd>Lenguaje de marcado de hipertexto</dd>

        <dt>CSS</dt>
        <dd>Hojas de estilo en cascada</dd>

        <dt>JS</dt>
        <dd>JavaScript, lenguaje de programación para la web</dd>
    </dl>

</body>
</html>

En el bloque anterior se muestran ejemplos prácticos de los tres tipos de listas en HTML: <ul> para listas no ordenadas, <ol> para listas ordenadas, y <dl> para listas de definiciones.
