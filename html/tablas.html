Elementos de Tablas en HTML

Elemento Descripción Ejemplo
<table> Define una tabla en HTML. Es el contenedor principal de las filas y columnas de datos. <table>...</table>
<tr> Representa una fila de la tabla, y dentro de esta fila se agrupan las celdas de la tabla. <tr>...</tr>
<th> Define una celda de encabezado en una tabla. Usualmente se utiliza para nombrar las columnas o filas, destacando el contenido con un formato especial (negrita y centrado por defecto). <th scope="col">Nombre</th>
<td> Representa una celda de datos en una tabla. Puede incluir texto, números, imágenes, entre otros. <td colspan="2">Datos</td>
<thead> Sección de la tabla que agrupa las filas del encabezado. Normalmente se utiliza para contener las celdas de encabezado de la tabla. <thead>...</thead>
<tbody> Sección del cuerpo de la tabla que contiene las filas con los datos principales de la tabla. Es donde se muestran los registros de información. <tbody>...</tbody>
<tfoot> Sección del pie de la tabla que se usa para agrupar los pies de las columnas o para colocar totales. Se suele utilizar al final de las tablas con cálculos. <tfoot>...</tfoot>
<caption> Define el título o descripción de una tabla. Se coloca antes de las secciones <thead>, <tbody> y <tfoot>. <caption>Título de la tabla</caption>

Etiqueta <table> para crear tablas

La etiqueta <table> se utiliza para definir una tabla en HTML. Esta etiqueta contiene otras etiquetas que definen filas y celdas dentro de la tabla.

Ejemplo básico:

<table>
    <tr>
        <th>Encabezado 1</th>
        <th>Encabezado 2</th>
    </tr>
    <tr>
        <td>Dato 1</td>
        <td>Dato 2</td>
    </tr>
</table>
Encabezado 1 Encabezado 2
Dato 1 Dato 2

Atributos principales de <table>:

  • border: Define el grosor de los bordes de la tabla (en píxeles).
  • cellspacing: Define el espacio entre las celdas.
  • cellpadding: Define el espacio dentro de las celdas, entre el contenido y el borde.
  • width: Define el ancho de la tabla.

Ejemplo con bordes, espacio entre celdas y celdas con padding:

<table style="border:1px solid" cellspacing="5" cellpadding="10">
    <tr>
        <th>Encabezado 1</th>
        <th>Encabezado 2</th>
    </tr>
    <tr>
        <td>Dato 1</td>
        <td>Dato 2</td>
    </tr>
</table>
Encabezado 1 Encabezado 2
Dato 1 Dato 2

Etiqueta <tr> para filas

La etiqueta <tr> define una fila dentro de la tabla.

Ejemplo:

<tr>
        <td>Celda 1</td>
        <td>Celda 2</td>
    </tr>

Etiqueta <th> para encabezados de la tabla

La etiqueta <th> define una celda de encabezado. Los encabezados están generalmente en negrita y centrados por defecto.

Ejemplo:

<th>Encabezado</th>

Etiqueta <td> para celdas de datos

La etiqueta <td> define una celda de datos dentro de una fila. Es el contenido principal de la tabla.

Ejemplo:

<td>Dato</td>

Ejemplo completo de tabla:

<table style="border:1px solid" cellspacing="2" cellpadding="5">
    <caption>Tabla de ejemplo</caption>
    <thead>
        <tr>
            <th>Nombre</th>
            <th>Edad</th>
            <th>Ciudad</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Ana</td>
            <td>28</td>
            <td>Madrid</td>
        </tr>
        <tr>
            <td>Juan</td>
            <td>35</td>
            <td>Barcelona</td>
        </tr>
        <tr>
            <td>Pedro</td>
            <td>22</td>
            <td>Valencia</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td colspan="3">Fin de la tabla</td>
        </tr>
    </tfoot>
</table>
Tabla de ejemplo
Nombre Edad Ciudad
Ana 28 Madrid
Juan 35 Barcelona
Pedro 22 Valencia
Fin de la tabla

Notas importantes:

  • caption: Se utiliza para agregar un título a la tabla.
  • thead, tbody y tfoot: Estas etiquetas se utilizan para agrupar las secciones de la tabla (encabezado, cuerpo y pie).
  • colspan: Atributo que permite que una celda ocupe varias columnas. Ejemplo: <td colspan="2">Celda amplia</td>
  • rowspan: Atributo que permite que una celda ocupe varias filas. Ejemplo: <td rowspan="2">Celda que ocupa dos filas</td>
